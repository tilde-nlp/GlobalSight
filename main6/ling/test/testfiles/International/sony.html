PCGA-DVD51をお使いのお客様へ
（2000/10/02）

PCGA-DVD51は Windows 98 Second Edition , Windows 2000 に対応したポータブルDVD-ROMドライブですが、下記に該当するバイオノートをお使いの場合は、DVD再生ソフトウェアをアップグレードする必要があります。

【対象機種】
  • PCG-Z505CR/K
  • PCG-SR9C/K

(*) DVD再生ソフトウェアアップグレードにつきましては、VAIOカスタマーリンクにお問い合わせください。

【無償アップグレードサービスの申し込み方法】
お申し込み受付は、以下の手順で行わせていただきます。
電話のみでの受付となりますことをご了承ください。

  1. お手持ちのDVD-ROMドライブの型名とシリアル番号をお確かめください。アップグレードの対象となる本体製品は、PCGA-DVD51です。
    • 本体製品のシリアル番号(15桁)を、メモなどに控えておいてください。
      シリアル番号はDVD-ROMドライブの底面に記載されています。

  2. お手持ちのノートブックコンピューターの型名をお確かめください。アップグレードの対象となる本体製品は、PCG-Z505CR/K、PCG-SR9C/Kです。
    • 本体製品のシリアル番号（15桁）を、メモなどに控えておいてください。
      シリアル番号は「Sony Notebook Setup」ユーティリティプログラムの「システム情報」タブに表示されます。

  3. Windows を起動し、[スタート]-[VAIO]-[ノートブックユーティリティ]-[Sony Notebook Setup] とメニューをたどってユーティリティプログラムを起動してください。

  4. お電話(*)でソニー(株)VAIOカスタマーリンクへ、お申し込みください。
    「PCGA-DVD51 付属ソフトウェアのアップグレード申し込み」とお申しつけください。お電話をいただいた際に以下の項目を確認させていただきます。

    • お客様のお名前（カスタマー登録された方はカスタマーIDをお願いいたします。）
    • お客様のご住所（発送先）、電話番号
    • ノートブックコンピュータのシリアル番号（15桁）
    • DVD-ROMドライブのシリアル番号（15桁）

以上の確認がすめば、お申し込みは完了いたします。

(*) 電話番号は、製品同梱の各取扱説明書の裏表紙、「VAIOサービス・サポートのご案内」または Windows の「システムのプロパティ」の「サポート情報」でご覧になれます。

ご注意事項
お申し込みをされる場合は、あらかじめ以下の内容についてご了承をいただきます。
  • お申し込みは1回限りとさせていただきます。
  • 着荷日時の指定はできません。ご了承ください。
  • ご提供するソフトウェアは、著作権によって守られています。
    お客様が複製を配布したり、ソニーが定めた用途以外に使用することは許諾されていません。
