Install / Uninstall

After setting GlobalSight server and adding patch, you can install/uninstall them.

Select patch

In the left tree window, there are two kinds of patches, new patches and installed patches. The new patches list the patches that are newly added into the patch installer and waiting for being installed. The installed patches list the patches that have been installed on the selected server and can be selected to uninstall.

Install

To install a new patch, please select a patch in the new patches list. Clicking the install button, Patch Installer will check related information and start installing the patch on the server.

Uninstall

To uninstall a patch, please select a patch in the installed patches list. Clicking the uninstall button, Patch Installer will check related information and start uninstalling the patch from the server.
