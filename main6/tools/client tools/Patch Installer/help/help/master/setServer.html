Set GlobalSight Server

Patch Intaller finds the server that was used last time and sets it as default path. You can also manually change it by "Set GlobalSight Server" option.

Please follow the steps below to configure the GlobalSight Server.

  • Click "File -> Set GlobalSight Server" on the menu, or use the shortcut key "Alt + S".
  • Select the GlobalSight server path in the folder selector.

Note: The correct server path should be "C:\\GlobalSight\" if your server is located at "C:\\GlobalSight\\jboss\\server..."
