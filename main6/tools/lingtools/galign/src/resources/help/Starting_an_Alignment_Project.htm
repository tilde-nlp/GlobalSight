Starting an Alignment Project

Step 1: Ensure that you have an Alignment Project to work on.  This package will have the extension .alp or .alp.zip.  if the extension is .alp.zip., rename it to .alp.

 

Step 2: From the File menu, select Open Project.  Browse to the .alp file you wish to open.

 

Step 3: Using the folder browser on the left of the screen, select a folder under "Alignments".

 

Step 4: From the Align menu, select Open Alignment.

 

Congratulations, you are ready to begin Alignment.  Continue with Aligning Segments.
