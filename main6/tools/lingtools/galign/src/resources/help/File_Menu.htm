File Menu

Open Project

Open an Aligner Project.  The project file must have the extension ".alp".

Close Project

Close the currently open project.  If there are unsaved changes you are prompted to save them.

Prepare for Upload...

Prepare the current project for upload.  If there are unsaved changes you are prompted to save them.

(File List)

Lists all recently opened Aligner Projects.

Exit

Closes the File Aligner.
