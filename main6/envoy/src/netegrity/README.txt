The JSP wbLogin.jsp should be used by WorldBank to login to System4.
This page should be protected by Netegrity and run in a separate web server.

The anonymousVendorViewer.jsp should be used by WorldBank for anonymous
read/only access to Vendor information.  This page should be protected by
Netegrity and run in a separate web server.

Only wbLogin.jsp and anonymousVendorViewer.jsp should be given to WorldBank.
The wbLoginTester*.jsp files are for internal testing
and development to simulate Netegrity.
