Browse Termbase

Summary:

Query the termbase to find a particular term or a group of terms.

Source

Select the language to search for the text entered into Query text box.

Target

Not active at this time.

Query

Enter the text to search on:

Use ! in front of your text to find the exact text entered.

Use # in front of your text to find fuzzy matches for the text entered.

Use * as a wildcard for more than one character.

Use ? as a wildcard for any one character.

Execute

Runs the query and displays the results under Terms Found.
