Select Files To Upload

Summary:

Select the files to upload for localization.

Left Panel

Contains a browser showing all folders visible from your computer.  Click a folder to display the contents of the folder in the upper right panel.  Double click on an icon to open the folder.

Upper Right Panel

Shows the contents of the folder selected in the left panel.  Select a file by clicking on it.  Use Ctr-Click to select multiple files.  Use Shift-Click to select a continuous list of files.

Lower Right Panel

Displays all files that will be uploaded.  Select a file by clicking on it.  Use Ctr-Click to select multiple files.  Use Shift-Click to select a continuous list of files.

Buttons

 

Adds the selected file to the list of files to be localized.

Removes the selected file from the list of files to be localized.

Cancel

Cancels content upload.

Previous

Return to the previous screen (Basic Information)

Upload

Upload the selected content to GlobalSight.  A progress bar indicates how far the upload has progressed.
