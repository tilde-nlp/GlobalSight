GS-Tagged Source Page Example

Example

 

<HTML>

<HEAD>

<TITLE>Test Page for GS Tags</TITLE>

</HEAD>

<BODY>

<H1>Test Page for GS Tags</H1>

<H2>My List</H2>

<OL>

<LI>List item text.

<LI>List item text.

<LI>List item text.

<GS ADD="LIST_ITEM"/> <!-- You can add a list item here. -->

<P>Paragraph text for paragraph 1.

<GS ADD="PARAGRAPH"/> <!-- You can add a paragraph item here. -->

<P>

<TABLE>

<TR>

<TD>Table Text</TD> <!-- You can add a table here. -->

</TR>

</TABLE>

<GS ADD="TABLE"/>

</BODY>

</HTML>
