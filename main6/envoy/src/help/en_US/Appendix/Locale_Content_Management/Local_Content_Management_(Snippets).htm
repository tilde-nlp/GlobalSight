Local Content Management (Snippets)

The Local Content Management feature lets you add and delete sections of HTML (sentences and formatting) for a particular target locale. This allows you to create target pages that have entire sections that do not appear in the source pages. This flexibility lets you to break the one-to-one, sentence-to-sentence relationship of source and target pages.

 

The sections or fragments of HTML that you can add or delete from target pages are called snippets. This section describes how to create, modify, delete, and manage snippets.

Enabling the Local Content Management Feature

To activate the Local Content Management, you must set Local Content Management equal to true during the installation of GlobalSight.

Creating Snippets

Create snippets by importing Snippet Import Files on the Snippet Import screen, located under the Setup menu. This populates the Snippet Library, which you will access later, in the Main Editor, in order to add snippets to target pages.

 

Adding Snippets to Target Pages

To add snippets to target pages:

  1. Create a Snippet Library by importing Snippet Import Files from the Snippet Import screen.

  2. Import a GS-tagged Source Page. The GS-tagged Source Page must have GS tags that correcspond to the snippets in your Snippet Library. See the GS-Tagged Source Page Example for reference.

  3. Open the page in the Main Editor and click on the Snippet Editor link, which will open the Snippet and Content Editor

  4. Use the Snippet and Content Editor to add or delete content from the page for the target locale of the Activity you are in.

 

Deleting Snippets from Target Pages

To delete snippets from target pages:

  1. Open the target page in the Main Editor

  2. Click on the Snippet Editor link

  3. In the Snippet and Content Editor, choose sections of the file that are marked as deletable

  4. Click Delete.

 

Exporting Target Pages with Snippets

Export target pages with snippets that you have added or deleted by following the normal export process for a file.

Exporting Source Pages with Snippets

If you want to re-import a source page of if you want to edit a source page without losing all the snippet work you have done on the page in GlobalSight, you can export the source page with all the snippets (GS tags) that you have added or deleted. You can do this by clicking on the Export Source button on the My Jobs screen.

Locale Manager Group

We have created a special user group in GlobalSight, name Locale Manager, whose purpose is to manage target pages that have locale content (snippets). Users who are Locale Managers have system privileges in between Localization Participan and Project Manager. A Locale Manager can import, translate, and export files just like a Project Manager, but only for locales defined in the roles. You defined their roles in the Create User process.
