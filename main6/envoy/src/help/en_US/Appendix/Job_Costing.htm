Job Costing

The GlobalSight Job Costing feature allows Project Managers to calculate the cost of localization jobs in a flexible and comprehensive way by providing in-progress cost estimation, surcharge additions, and final cost overrides. This section explains steps required to implement Job Costing in GlobalSight.

 

  1. Activate Job Costing During Installation
    During the installation of GlobalSight, set the "Enable Job Costing" parameter to "true" to activate GlobalSight Job Costing.
     

  2. Create Rates
    Create Rates that are appropriate for you costing needs. There are four types of Rates:

    1. Word Count

    2. Hourly

    3. Page

    4. Fixed
       

  3. Create a Workflow with Activities that uses the Rates you defined above.
     

  4. Create a Localization Profile that uses the Workflow you defined above.
     

  5. Create a Job.
     

  6. Enter the estimated pages and the estimated hours for the job.
    As a Project Manager, you may want to estimate the cost of a job before the job is completed or even before work has begun on the job. You can do this as follows:

    1. You can manually enter the estimated number of pages in the job from the Job Details screen by clicking on the Pages > Edit link. The default number of pages is determined by the GlobalSight extraction process. The reason that you may want to manually enter the number of pages is that for long Microsoft Word documents, for example, one multi-page document will become one single page in GlobalSight since GlobalSight ignores Microsoft Word page breaks. Therefore, you may want to enter in the Microsoft Word page count for accuracy.
       

    2. You can enter the estimated number of hours for each Activity in a workflow by opening the graphical workflow screen, right-clicking on each Activity and entering the number of hours.

      Note:       The number of pages and hours that you enter in this way will be stored and used to calculate the Estimated Cost of the job. The Actual Cost of the job will be calculated by taking the values that Localization Participants enter when then complete their Activities.
       

  7. Start work on the job.
    As Localization Participants work on their Activities, they will use the Activity Details screen to enter in actual values for page pages and hours. These values will go into the Actual Cost calculation.
     

  8. Add Surcharges to the job.
    At any point in the job, you can add Surcharges from the Job Details screen. Surcharges are non-rate-based charges. There are two types of Surcharges:

    1. Flat

    2. Percentage
       

  9. Finish the job.
     

  10. Final Cost
    GlobalSight calculates the Final Cost by adding the Actual Cost and the Surcharges. You can override this Final Cost by clicking the Final Cost  > Edit link and entering any Final Cost they choose. This Final Cost then becomes an Override Final Cost, meaning it  has not been "calculated", rather it has been entered manually. You can remove the override cost and revert the Final Cost to a calculated cost by checking the Remove Override checkbox on the Edit Final Cost screen.
