Customizing Segmentation Settings

You can customize GlobalSight to segment at the sentence level or the paragraph level by editing the file <weblogic_home>/globalsight/properties/Diplomat.properties.

 

To segment data at the sentence level (default), set the "segment" property equal to "sentence". For example:

segmentation=sentence

 

To segment data at the paragraph level, segment the "segment" property equal to "paragraph". For example:

segmentation=paragraph

 

Customers with data from GlobalSight.4.2 should ensure that segmentation_preserve_whitespace = false, to ensure proper leveraging of legacy GlobalSight data.  Customers with a newer installation of GlobalSight should set this parameter to true to ensure proper segmentation of segments containing characters that are treated as whitespace.

 

You need to restart Weblogic for the changes to take effect.
