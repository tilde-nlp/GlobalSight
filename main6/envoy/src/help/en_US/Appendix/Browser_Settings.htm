Browser Settings

If you are having trouble opening certain individual segments in the pop-up editor, it is recommended that you make the following changes to your Internet Explorer settings:

 

Select Internet Options from the Tools menu.  Click the advanced tab and look for the following options:

 

Under Java (Sun):

Use Java 2 v1.3.1_04 for <applet> (requires restart)

This item should be unchecked.  

 

Under Microsoft VM:

Java console enabled (requires restart)

This item should be checked.  

 

Java logging enabled

This item should be unchecked.  

 

JIT compiler for virtual machine enabled (requires restart)

This item should be checked.
