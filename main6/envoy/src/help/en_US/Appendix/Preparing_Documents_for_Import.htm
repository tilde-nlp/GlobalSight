Preparing Documents for Import

  1. Create the CXE document directory that you entered during the GlobalSight Files installation above. For example:
    d:\GlobalSight\docs
     

  2. Create a subdirectory where you will store your source files, for example:
    d:\GlobalSight\docs\en_US
     

  3. Put the documents you want to import in this directory.
     

  4. If your source language is different that en_US then replace en_US with your source language locale, for example, es_MX
     

  5. Login to GlobalSight and import the files.
