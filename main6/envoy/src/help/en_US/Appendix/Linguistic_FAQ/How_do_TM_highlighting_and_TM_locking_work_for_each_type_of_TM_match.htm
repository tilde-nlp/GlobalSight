Definition of terms for the table.

LGEM

Leverage Group Exact Match. An LGEM is an Exact Match from the same page. See description above.

Exact Matches

Exact Matches are TM matches that have the same text, case, whitespace, line breaks, and formatting.  

Demoted Exact Matches

Demoted Exact Matches are TM matches that have the same words, but different spaces of formatting. Demoted Exact Matches are also TM matches where the source that is matched has two or more different translations.

Fuzzy Matches

Fuzzy Matches are TM matches where the source segments being matched have different words, spaces, and formatting.

Locked

Locked segments are segments that are read-only in the Online Editor and ignored on upload when translating offline.

Highlighting

TM highlighting is the background color that GlobalSight applies to segments that are matched in TM.

Formatting preserved

Fuzzy Matches often match in text but not in formatting (e.g.. bold, italic, underline). GlobalSight returns the match with as much formatting as possible based on the match score.
