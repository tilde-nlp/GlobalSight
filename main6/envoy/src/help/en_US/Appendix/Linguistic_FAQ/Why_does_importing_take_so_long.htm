Why does importing take so long

Importing in GlobalSight takes can take long time because this the step where the raw data is prepared for use in the system. This preparation includes processing the data in memory and insertion of the data into the database. The burden of this processing is put on the Import step so that the other time-critical system operations -- Editing, Job Advancement, Exporting -- can happen quickly.

The import process has the following steps:

  • Extraction - Separating translatable data from non-translatable data
  • Segmentation - Breaking paragraphs into sentences
  • Word Count
  • Save the data to the database
  • Creating Source pages from GXML
  • Leveraging
  • Job Creation
