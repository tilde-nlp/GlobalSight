How are TM Fuzzy Matching scores calculated

The score algorithm is: 2C/(A+B)

A = number of words in segment 1

B = number of words in segment 2

C = number of common words between the two segments

 

For example:

Segment 1: This is a dog.

Segment 2: This is a big dog.

The score would be: 2(4)/(4 + 5) = .89

Write .89 as a percentage and you get 89%.

 

Whitespace, case, and formatting differences are also factored into the fuzzy score by assigning penalties for such differences. For example, if everything matches (i.e., text, case, and whitespace) except for formatting (i.e., bold, italic), then we just assign a penalty of -1. So the total score would be 99%.
