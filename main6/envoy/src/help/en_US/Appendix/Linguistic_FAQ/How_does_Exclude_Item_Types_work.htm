How does Exclude Item Types work

The Exclude Item Types feature lets Project Managers exclude certain types of segments from translation. For example, if a Project Manager does not want any of the JavaScript in a page to be translated, then they will enter the Item Type for JavaScript into the Exclude Item Types of the Localization Profile. The translator will then not be able translate any of the JavaScript in the page.

 

The Item Types are entered into the Localization Profile as a pipe-delimited list. The available Item Types are listed in the documentation.
