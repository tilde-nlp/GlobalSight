How does GlobalSight decide how many Fuzzy Matches to display to the user

Fuzzy Matches are like search results in a search engine, the number of hits can vary widely depending on the the search string and the data being searched. By default, GlobalSight discards matches that are lower than a 70% score. This value is customizable from the System Parameters screen. After this matching threshold is applied, the system only displays the top three matches that are returned. This setting is based on industry standards and is not customizable.
