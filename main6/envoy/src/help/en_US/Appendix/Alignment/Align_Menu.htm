Align Menu

Open Page

Opens a pair of files to be aligned.  Select a sub-folder under the Alignments folder in the folder tree on the left.

Save Page

Saves the currently open alignment work.

Close Page

Closes the currently open alignment.

Remove Page

Removes the currently open alignment work from the project.  Note: You cannot undo this action.

Approve All Alignments

all of the likely matches, indicated by a dashed line between source and target.  You should still review the alignments and adjust as necessary.
