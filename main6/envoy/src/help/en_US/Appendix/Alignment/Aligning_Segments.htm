Aligning Segments

If you have followed the instructions under Starting an Alignment Project, you should see segments in the alignment window.  The system indicates likely matches by drawing a dashed line between the source and target matches.  

 

To confirm a match, use one of the following two methods:

 

Method 1: Click the document icon next to the source segment and drag your cursor to the document icon next to the target icon.  When you are successful, the document icons will appear like this: .  A solid line will connect the source and target document icons.  You may also click the icon next to the target segment and drag it to the source segment.  It works equally well in both directions!

 

 

Method 2: Select either the source or target document icon and right click.  From the context menu that appears, select Approve Alignment.  This will only work if the system has indicated a likely match by drawing a dashed line between source and target.

 

 

Method 3: From the Align menu, select Approve All Alignments.  This approves all of the likely matches, indicated by a dashed line between source and target.  You should still review the alignments and adjust as necessary.

 

 

To break an alignment between a source and target segment:

 

Use Method 2 described above.  From the context menu, select disconnect alignment.

 

 

To match a source segment to more than one target segment:

 

Use Method 1 described above.  Click the document icon next to the source segment and drag it to the first target segment.  Click the document icon again and drag it to the second target icon.

 

 

To match a target segment to more than one source segment:

 

Use Method 1 described above.  Click the document icon next to the first source segment and drag it to the target segment.  Click the document icon next to the second source segment and drag it to the same target icon.

 

To line up source and target segments across from one another:

 

Right click on the document icon next to the segment and select Line up Segments from the context menu that appears.

 

To remove a segment from the alignment:

Note: You cannot undo this action.

 

In some cases it may be helpful to completely remove segments which do not have a match.  Disconnect any alignment lines, if necessary, then right click on the document icon next to the segment and select Remove Segment from the context menu that appears.

 

 

Next Steps:

 

Remember to save often (from the Align menu, select Save Page).  When  you have finished aligning the files, you can continue to Uploading Alignment Work to GlobalSight.
