Working Offline

See also: How to save a file for upload.  The information below only applies to files in List View with numbered segments (extracted files).

 

Segment Resources

In List View, segment resources provide additional information about each segment such as the Original Source text, TM Matches and Terminology Matches. You can navigate to each of these resources by using a combination of hyperlinks and/or annotations which appear in the download file. In addition to this, all resources may be viewed by opening the index.html in your web browser. The index.html file is located in the root of your project directory.

 

Note: resources links will not appear in the download file if you select plain text as the download file format. However, all resources can still be viewed by opening the index.html directly.

 

Example - Resource Annotations

# Resources: [Source22], [TM23], [Terms24],  

 

[Source(n)]

An annotation containing a copy of the original source text.

(n) is an internal id generated by the host editor.

 

[TM(n)]

An annotation containing a list of fuzzy matches for the source text.

(n) is an internal id generated by the host editor.

 

[Terms(n)]

An annotation containing a list of terminology matches found in the source text.

(n) is an internal id generated by the host editor.

 

Example - Resource Hyperlink

# Resources: Source, TM, Terms  

 

This hyperlink may appear in place of annotaions if the download file becomes very large. Using this link, you can navigate directly to the appropriate entry in the segment resource pages. Here the Original Source text, TM matches, Terminology Matches and placeholder mappings can be found.
