How to Save Files for Upload

For an Extracted File:

Upload the RTF or Text file that you have edited.  There is no need to pre-clean Bilingual Trados files before upload. They will be cleaned automatically.

 

For a Secondary Target File:

Make sure that you have downloaded the file first from the download screen.  Either edit this file, or make sure that the file you have edited has the same filename.

 

For an Unextracted File:

Make sure that you have downloaded the file first from the download screen.  Either edit this file, or make sure that the file you have edited has the same filename.
