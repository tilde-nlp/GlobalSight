Tips For Working With Large RTF Files

Save the file in DOC format.

For convenience, you may save the RTF files you downloaded as DOC files while you are working on them. In most cases, the RTF files generated by the system are manageable in RTF format. However, if a large RTF file takes too long to open, you may save it in DOC format after you have opened it for the first time in RTF. Thereafter, the DOC file will load into your editor in just a few seconds.

 

If you SaveAs DOC, you must be careful to continue work in the DOC file (instead of the original RTF file). When you finish work, you must save as RTF before upload.
