Document Check

Before uploading, run the "Check Document" command from the GlobalSight menu. Checking errors before uploading is more efficient and reduces the time it takes to correct errors.

 

Any errors are displayed in the Error Navigator. The Error Navigator assists the user in moving to the given segment and by positioning the cursor at the proper location to insert a missing segment.

 

Clicking "Next Error" button causes the Segment Editor to move to next error location.

 

Placeholder Tag Errors

For details on placeholder errors, see Working with Placeholder Tags.

 

Missing Segments

If a segment is found to be missing (deleted), you will be prompted to copy the segment back to its original location.  Click "Insert missing segment."  You may then need to enter a translation for this text.  

 

If loose text (text not surrounded by a segment bookmark) is found at this same location, adjust the auto-selection (if necessary) to include only the loose text that belongs to the segment and click "Attach Source to the selected Target text."  This places the segment bookmarks around the loose text, restoring the segment.
