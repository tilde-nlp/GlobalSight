Segment Protection

In order to ensure that the segmentation remains intact, certain operations in Word are protected against.  If you try to perform one of the protected operations, a warning message appears.

 

The following operations are protected if they result in a segment delimiter (a bookmark) being deleted:

  • Copy/Paste

  • Cut

  • Delete key

  • Backspace

 

Recommended: Use the Segment Editor's Open and Close functions to edit segments. Using these functions enables access to TM and Term resources and ensures that segments are not corrupted. If you use the Segment Editor, you may turn off the visible bookmarks to make the document less cluttered. However, if you are going to edit the document directly without using the Segment Editor, it is highly recommended to leave the bookmarks turned on so that the segment delimiters can be seen while editing directly within the text.
