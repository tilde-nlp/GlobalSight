Segment Editor

Each segment (sentence) is separated by delimiters (bookmarks) to ensure the best possible translation memory from the translated file.  It is recommended that you leave bookmarks turned on so that the segment delimiters can be seen and are not inadvertently deleted. If you always use the Segment Editor to edit segments, you may turn bookmarks off.

 

Use the commands under the GlobalSight menu to open individual segments for editing.  Or use the GlobalSight Toolbar.

 

Toolbar

You can access the GlobalSight toolbar by right-clicking on the Microsoft Word toolbar and making sure the GlobalSight item is checked.  The toolbar may be obscured by other toolbars, to the far left of the other tool bars.  Drag the move handle on the toolbar to place it where you can see all of the buttons.  Hold you mouse over a button to view the function of that button.

Open (Alt-Ins)

Opens the segment for editing.  Open or Open/Get is the recommended way of editing segments as it ensures that the segment separators are not corrupted or lost during the process.  You can also directly edit the text by simply typing over it.  However, you must be careful not to delete bookmarks and you will not see translation memory or terminology matches unless you open the segment.

 

Open/Get (Alt-Ctrl-Home or Alt-[ or Alt-Home)

Opens the segment and retrieves the highest possible fuzzy match.

 

This Next Fuzzy (Alt-N)

Displays the next fuzzy match for the current open segment.  Use this option to cycle through all available fuzzy matches for the open segment.

 

Set/Close (Alt-Ctrl-End or Alt-] or Alt-End)

Saves changes and closes the current open segment.  You must still save your document periodically.

 

Set/Close Next no 100% Open/Get (Alt-\ or Alt-Ctrl-Num+)

Saves changes and closes the current open segment.  Opens the next segment that is not an exact match.

 

Close (Esc)

Closes the current open segment without saving changes.

 

Restore Source (Alt-Delete)

Copies the source text into the currently selected segment.  The segment does not have to be open for this operation.

 

Show Segment Name/Boundary (Alt-S)

Highlights the segment at the current cursor position.

 

Merge (Alt-Ctrl-PageDn)

With Previous: Merges the current segment with the previous segment.  The current segment is one that is currently open, or the segment at the current cursor position.  The new merged segment is opened for editing.

With Next: Merges the current segment with the next segment.  The new merged segment is opened for editing.

Cancel: Cancels the Merge operation.

 

Split (Alt-Ctrl-PageUp)

This operation only works on previously merged segments.

Head:  The first segment of the merged group is split from the group.  The original source text or any exact match is copied into the segment. The text in the remaining segment (group) remains unchanged.

Tail:  The last segment of the merged group is split from the group.  The original source text or any exact match is copied into the segment.  The text in the remaining segment (group) remains unchanged.

Cancel: Cancels the Split operation.

 

Terminology (Alt-T)

Opens the terminology window for this document.  When a segment is open, only terminology matches for that segment are displayed.  Double-Click any terminology entry to copy it into the text.  This window remains open until you close it.

Next: Displays terminology matches for the next segment.

Previous: Displays terminology matches for the previous segment.

Synch: Displays terminology matches for the currently selected segment.

Dim Tags (Alt-D)

Placeholder tags in this document are colored gray to make them less obtrusive.

 

Brighten Tags (Alt-B)

Placeholder tags in this document are colored light blue to make them more visible.

 

 

Details...

 

Opens the placeholder detail window for this document.  When a segment is open any placeholders for that segment are displayed.  The code that the placeholder represents is also displayed.  You can also double click on any placeholder tag to display the details window.

Next: Displays placeholder details for the next segment.

Previous: Displays placeholder details for the previous segment.

Synch: Displays terminology details for the currently selected segment.

Check Document...

Before uploading, be sure to check the document.  The error checker will look for the following conditions in the document:

  • Missing placeholder

  • Invalid placeholders

  • Placeholder order

  • Missing segments

  • Embedded or overlapping segment separators

  • Loose text (text not contained between any segment separators)

 

See Document Check for more details on this functionality.

 

Enable all Pop-up Warnings

Protection warnings appear in popup message boxes. Each message can be disabled after it appears.  Disabled messages are still displayed in the application status bar. This command re-enables all popup messages.
