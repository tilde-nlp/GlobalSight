Overview

In the Inbox directory, you will find files for translation.  One file for translation appears for each file in the Job.  Secondary Target Files can be translated directly in the native application in which they were created (e.g. Quark, FrameMaker, etc.).  Primary target files are in RTF format and contain individual segments grouped together by paragraph.  This is in contrast to list view where each segment is separated by header text.  In paragraph view, segments are separated by simple bookmarks to minimize the impact and ensure the integrity of the TM automatically built on upload.

 

This help system contains individual pages specific to different aspects of translating these paragraph mode files.

 

Startup - General information about opening files

Segment Editor - Opening, closing, editing and saving segments

Segment Protection - Bookmarks and segment integrity

Colors - A key to color coding used for segments

Working with Placeholder Tags - Protecting inline code and segment formatting

GlobalSight Options - GlobalSight options within Word
