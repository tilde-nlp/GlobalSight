Startup

GlobalSight uses document templates to provide numerous features for translating files offline.  When Primary Target Files are opened, the system checks to see if other templates containing macro commands that may damage the GlobalSight file are installed.  If so, these templates must be temporarily disabled, however, they will be re-enabled the next time Microsoft Word is started.

 

When opening especially large GlobalSight Documents containing a reference to a lot of TM or Term data, a message may appear stating that the TM and Term resources files need to be optimized. This action normally occurs automatically the first time a resource is requested. However, if the resource files are especially large, GlobalSight notifies the user and then performs the operation immediately. In this case, optimizing the resource files takes only a few minutes as the associated resource files are converted from RTF to DOC format - thus providing significant performance benefits when retrieving resources.

 

Secondary Target Files and Un-extracted Primary Target files should be edited in their native application.  No translation assistance features are provided for these files.

 

To begin working, unzip the downloaded archive for this activity.
