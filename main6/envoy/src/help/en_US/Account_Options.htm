Account Options

Summary

Additional user preferences for customizing Editor behavior.

General Options

 

Page Name Display:

In the Job Details screen, the name of the file is displayed:
full (with path): displays the name file, including the entire path relative to the import directory.
as filename with tooltip: displays only the name of the file.  The path is visible as a popup when mousing over the name.

Editor Selection

 

Edit Pages In:

Inline Editor: Text is edited inline in a traditional translation interface.
Popup Editor (Advanced): Text is edited from the following view modes: text, list or preview.

Main Editor Options

 

Normal Link Color

For use with HTML preview.  Translatable segments are linked, so if the background color of your pages is the same as the default link color, you may wish to override the defaults.  This selection determines the color for all links.

Active Link Color:

As above.  This selection determines the color for links as they are clicked.

Visited Link Color:

As above.  This selection determines the color for visited links.

Automatically unlock segments

Checked:  Exact matches are never locked.

Unchecked: Exact matches are locked, and must be unlocked before they can be edited.

Default Layout:

Determines the default layout for the main editor.  Select one of the listed options.

Default View Mode:

Determines the default view mode for the main editor.  Select one of the listed options.

Segment Editor Options

 

Automatically save segments:

(Popup Editor Only)

Checked: Segments are automatically saved after clicking close, or next/previous segment.  

Unchecked: You are prompted to save your changes after clicking close, or next/previous segment.

Automatically adjust trailing whitespace:

Checked: Trailing whitespace in certain segment types such as URLs and localizable script segments is automatically removed.

Unchecked: Trailing whitespace in these segment types is not removed.

Default PTag Mode:

Compact: By default placeholder tags are displayed in compact form.
Verbose: Placeholder tags are display in verbose form by default.

Buttons

 

Cancel

Discard changes and return to the previous screen.

Done

Save changes and return to the previous screen.
