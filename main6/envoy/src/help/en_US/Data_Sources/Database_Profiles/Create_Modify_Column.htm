Create/Edit Column Profile

These screens are only viewable if you have a database integration.

 

Summary:

Describe one of the columns necessary for localization.  This includes the column name, table name, and format.

Name:

Column name as it appears in the database.

Label:

A short description (label) for this table that is displayed to localization participants.

Table:

Table name as it appears in the database.

Mode:

Translatable - Select this if the contents of the column are translatable.

Contextual - Select this if the contents of the column are to be provided to translators as context, but are not to be translated.

Invisible - Select this if the contents of the column are necessary to identify the row (primary key) but are not to be translated or needed as context.

Format:

Select the Format of the column's data (see Supported Data Formats).

Note: Selecting the XML format produces an additional dialog: XML Rules.

XML Rules:

Select the XML rules to use to determine what is localizable within the XML content  (see also XML Rules, Writing XML Rules).

Buttons

 

Close

Cancels the column profile creation/edit.

OK

Saves the new/modified column profile.
