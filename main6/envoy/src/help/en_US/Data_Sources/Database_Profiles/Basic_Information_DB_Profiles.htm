Database Profiles Basic Information

These screens are only viewable if you have a database integration.

 

Summary:

Specify read/preview/write SQL statements for localizable information in database tables.

Name:

For display in GlobalSight. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional, viewable by administrators in GlobalSight.

Localization Profile:

Determines the workflow templates, target locales, and job dispatch settings for localization.

Buttons

 

Cancel

Cancels database profile creation/edit.

Previous

Cancels database profile creation/edit.

Next

Moves to the next screen of database profile creation (Database Columns Profiles).
