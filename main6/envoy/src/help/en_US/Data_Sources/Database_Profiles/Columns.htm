Database Column Profiles

These screens are only viewable if you have a database integration.

 

Summary:

Describe all of the columns necessary for localization of any given table.  This includes columns that are translatable, columns that provide context to translators, as well as the primary key columns.

Table

Displays database column profiles that have been added to the system.  Select a column profile by clicking anywhere in the profile's row.

Buttons

 

Move Up

 

Moves the selected column profile up in the list.

Move Down

Moves the selected column profile up in the list.

Cancel

Cancels the database profile creation/edit.

Remove

Remove the selected column profile.

Edit

Modify the selected column profile.

New

Add a column profile to the list.  Add all of the columns profiles necessary for localization of a single table (see Create/Modify Column).

Previous

Returns to the previous screen (Database Profiles Basic Information).

Next

Moves to the next screen (Database Read SQL).
