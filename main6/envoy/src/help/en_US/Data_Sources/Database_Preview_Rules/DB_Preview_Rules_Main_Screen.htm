Database Preview Rules Main Screen

These screens are only viewable if you have a database integration.

 

Summary:

Create instructions, written in XML, detailing the path to a template page for displaying source and target data dynamically during preview.  Requires DB Preview SQL to be filled out in the DB Profile.

Table

Displays all database preview rules that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a preview rule by clicking anywhere in the rule's row.

Buttons

 

Edit

Modify the selected DB preview rule (see Create/Edit DB Preview Rules).

New

Create a new DB preview rule (see Create/Edit DB Preview Rules).
