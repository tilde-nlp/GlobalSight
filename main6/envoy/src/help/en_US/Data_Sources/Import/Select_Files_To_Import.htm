Import

Summary:

Select the files to import from the /docs directory.  This list is limited to files with the extensions listed in the file profile.

(Folder Name)

Clicking on the name of a folder opens that folder, showing files that have an extension listed in the selected file profile.

Adds the selected file the Files Selected For Import list.  If a folder is selected, all files matching the extensions listed in the selected file profile in that folder and any subfolders are added to the Files Selected For Import list.

Removes the selected file or files from the Files Selected For Import list.

Moves up a directory.

Check All

Selects all files or directories displayed

Clear All

Un-selects all files or directories displayed

 

Special Note: A PDF format does not store words or sentences as a normal file does.  In order to import a PDF, OCR (optical character recognition) is used to first convert the file into a Word document.  When the file is exported, it is exported as a .doc (Word) file.  This After export, the .doc file can be used to generate a PDF.

Buttons

 

Cancel

Cancels the import.

Next

Moves to the next screen of import (Map Files).
