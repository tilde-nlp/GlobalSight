Create/Edit Connection

These screens are only viewable if you have a database integration.

 

Summary:

Specify a connection string and driver for reading/writing to a database.

Name:

For display in GlobalSight. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional, viewable by administrators in GlobalSight.

Database Driver:

The fully qualified java class path for the database driver used to communicate with your database.

Connection String:

Use the standard Java database connectivity definition

<DriverType>:<DatabaseType>:<ConnectionType>:@<server.domainname.doma

in>:<port#>:<DatabaseInstanceProcessName>

NOTE: The user name and password are not included in this string. They are entered in the User Name and Password fields below.

Oracle Example:

driver: oracle.jdbc.driver.OracleDriver

connection string: jdbc:oracle:thin:@customerdb.customer.com:1521:custdb

username: customer

password: password

User Name:

The database username for this database user.

Password:

The password for the user entered above.

Buttons

 

Close

Cancels the database connection creation/edit.

Save

Saves the new/modified database connection.
