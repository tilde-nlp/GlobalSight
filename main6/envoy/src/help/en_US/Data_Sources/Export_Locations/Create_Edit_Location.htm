Create/Edit Export Location

Summary:

Describe an export location

Name:

This name is for display and will be seen in the export screen when selecting a location. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Export Directory Location:

Enter the path name of the export location.  This location can reference any named drive that GlobalSight can see. Examples: Windows - c:\myfiles\myexportdir

Solaris - /var/myexportdir

Description:

Optional.  This is seen only on the export locations main screen.

Buttons

 

Cancel

Cancels the export location creation/modification.

OK

Saves the new/modified export location.
