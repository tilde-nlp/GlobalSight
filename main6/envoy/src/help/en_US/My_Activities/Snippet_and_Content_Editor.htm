Snippet and Content Editor

Summary:

Add to and delete snippets from the target page.  Optional

Page Name

The fully qualified name of the page you are editing.

View Mode

Allows you to view the page as rendered HTML (Preview) or as raw HTML (text).

Find:

Enter the text you wish to find on the page and press return or click the small triangle to begin searching.  If the first match is not what you are looking for, press F3 to highlight the next match.

Add Snippet Icon - Double click on this icon to add a snippet at this location in the page. This will bring up the Select Snippet dialog, which will allow you to choose the snippet that will be added at this location.

Deleted Snippet Icon - Double click on this icon to un-delete a snippet you have previously deleted at this location.

The dashed blue line indicates text that is deletable.  Double click on this box to remove the text inside.  Deleted text is replaced by a minus sign in a blue circle (see above).  Deletes can be nested, so if you wish to delete a large section, rather than deleted several small sections individually, check to see if there is a blue dashed line that encompasses the entire section.

 

Note: Due to the table structure of some pages, some text may not show as deletable in the Preview view mode (see View Mode above).  Select Text as the View Mode and check to see if the text can be deleted.
