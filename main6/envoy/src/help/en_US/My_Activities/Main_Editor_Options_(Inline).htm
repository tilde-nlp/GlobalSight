Main Editor Options (Inline)

Summary

Additional user preferences for customizing Main Editor behavior

Automatically adjust trailing whitespace:

Checked: Trailing whitespace in certain segment types such as URLs and localizable script segments is automatically removed.

Unchecked: Trailing whitespace in these segment types is not removed.

 

Automatically unlock segments

Checked: Locked segments appear unlocked by default.

Unchecked: Locked segments must be unlocked by clicking on the unlock button.

Show machine translation (MT) button

Checked: Machine translation button (MT) appears in the editor toolbar.

Unchecked: Machine translation button (MT) does not appear in the editor toolbar.

Highlight PTags:

Checked: Placeholder tags are highlighted to assist in viewing.

Unchecked: Placeholder tags are not highlighted.

Buttons

 

Apply

Apply the change and close the window.

Close

Close the window.
