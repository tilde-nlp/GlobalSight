Segment Editor Options

Summary

Additional user preferences for customizing Segment Editor behavior.

Automatically save segments:

Checked: Segments are automatically saved after clicking close, or next/previous segment.  

Unchecked: You are prompted to save your changes before moving to the next segment or clicking close.

Automatically adjust trailing whitespace:

Checked: Trailing whitespace in certain segment types such as URLs and localizable script segments is automatically removed.

Unchecked: Trailing whitespace in these segment types is not removed.

Default PTag Mode

Compact: PTags are displayed in short format, i.e. [l1] would represent the opening tag of a link.

Verbose: PTags are displayed in long format, i.e. [link1] would represent the opening tag of a link.

Highlight PTags:

Checked: Placeholder tags are highlighted to assist in viewing.

Unchecked: Placeholder tags are not highlighted.

Previous/next always iterates subs:

Checked: Clicking previous or next will show link URLs, image paths and other subflows of segments.

Unchecked:  Subflows as described above are skipped when previous or next is clicked.

Buttons

 

Apply

Save and apply changes.

Close

Cancel changes and close window.
