Segment Editor (Text)

Summary:

Localize an individual segment

Source

The source text for localization.

Target

Type your localization (translation/edit) in this box.

Terminology Matches

Click a match to copy it at the end of the target area; right-click to browse the termbase for this project (see Browse Termbase)

Match Results

Lists any fuzzy matches found in the TM for this segment.  Each match is listed with a percentage that indicates how close the matched source material is to the actual source.  Click a segment to copy it into the target field.
Click this icon to view the match result in original context via the Corpus Results.

 

Segment Versions

The segment as it appeared in each of the previous activities.

Save

Saves your work on this segment.

You may receive placeholder tag errors (see Placeholder Tag Errors).

 

Keyboard equivalent: Ctrl+S

Previous Segment

Moves to the previous segment to translate.

You will be prompted to save the segment you are on.

 

Keyboard equivalent: Ctrl+PageUp

Next Segment

Moves to the next segment to translate

You will be prompted to save the segment you are on

 

Keyboard equivalent: Ctrl+PageDown

Revert

Reverts to the last saved version of this segment.

Details

Contains troubleshooting information about a segment that may assist GlobalSight Customer Support.

Compact/Verbose Tags

Switches between compact and verbose display of placeholder tags (see Placeholder Tags).

x2 x2 [b] [i] [u] [br] [nbsp]

Insert any of these formatting tags into the text (see Placeholder Tags).

[ptag]

Opens a popup window displaying all placeholder tags for this segment.  Click a placeholder tag to insert it into the text.  See also Placeholder Tags.

Close

Closes this window.

You will be prompted to save the segment you are on.

Keyboard Equivalents

 

Use the following keyboard equivalents for faster Segment Editor navigation:

Ctrl+S: Save segment

Ctrl+PageUp: Previous Segment

Ctrl+PageDown: Next Segment

Shift+Ctrl+-: Subscript

Shift+Ctrl+=: Superscript

Ctrl+B: Bold

Ctrl+I: Italic

Ctrl+U: Underline

Ctrl+LeftArrow: Word left

Ctrl+RightArrow: Word right

Shift+Ctrl+LeftArrow: Select word left

Shift+Ctrl+RightArrow: Select word right
