Search Window

Summary:

Search the segments by User ID or SID

Searched by User ID

Search all the segments, if the latest modifier of segments equals to the selected User ID. Highlight the segments.

Searched by SID

Search all the segments, if some segment Sids equal to the selected SID. Highlight the segments.

Reset to Normal

Remove all highlight in Main Editor.

Filter By

If there are many User IDs and SID to selected, user can input the key words and click on "OK" button. Then the items including the key words can be filtered out.
