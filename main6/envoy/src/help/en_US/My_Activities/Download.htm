Work Offline - Download

See also Upload

 

Summary:

Download files to work offline.

Primary Target Files

 

Select Pages

Select the pages you wish to localize offline by clicking the checkboxes next to each file name.

You can click Check All or Clear All to select all or none of the files at once.

Download Options (for extracted files)

 

Format

Bilingual TRADOS® RTF - This option exports the content as a list of bilingual Trados segments.

RTF (list view) - This option exports the content as a list of individual segments.  This format is somewhat smaller and simpler than paragraph view, but lacks many of the built-in features that paragraph view offers.

Xliff 1.2 - This option exports the content in XLIFF 1.2 format. This is the recommended format for SDL Trados (Tag Editor) as well as other XLIFF-capable translation tools.

Xliff 2.0 - This option exports the content in XLIFF 2.0 format. This is the recommended format for SDL Trados (Tag Editor) as well as other XLIFF-capable translation tools.

OmegaT - This option exports the content in OmegaT format (XLIFF-capable) for OmegaT Editor.

Placeholder Format

For non-XLIFF formats, select compact or verbose tags as per your preference. When exporting to XLIFF format, this option has no effect.

TM Options

For XLIFF format, select TMX file 1.4b for a TMX 1.4b file, or Annotations to have TM matches inserted into the XLIFF file as alternate translations.

For RTF formats, select the method by which the downloaded RTF file displays links to Source segments, TM matches, and Terminology matches.

  • Annotations will insert the Resource links as Microsoft Word Comments (hightlighted in yellow).

Note: When using Word2002 you may not wish to use this option as the full text of an annotations is displayed within the text.

  • None will not insert any Resource links. You will have to manually go to the HTML resource pages in the download directory. This method is best if you want to minimize the size of the download package.

Penalized Reference TM Options

Only be active once either of the TMX File options for the TM Options dropdown is selected. For OmegaT exports, this option is not enabled/considered at all.

Separate TMX with Pre-Penalized Source - Selected by default. A separate TMX containing all content (with "penalized" source segments) from the penalized reference TMs only should be exported when checked.

Separate TMXs Per Reference TM Penalty - A separate TMX for each Reference TM penalty should be exported when checked.

Allow Edit of Locked Segments

Allows you to make changes to In Context Exact matches and Exact matches. If set to No, changes to In Context Exact matches and Exact matches will be ignored on upload. Both options allow you to see these matches.

This option only appears if your project manager has enabled it.

Populate 100% Target Segments

Checked (Default): 100% matched target segments will be populated into offline downloaded files such as Xliff, TTX files.
UnChecked: Source segments will be populated into offline downloaded files instead of matched target segments.

Populate MT Target Segments

Checked (Default): MT matched target segments will be populated into offline downloaded files such as Xliff, TTX files.
UnChecked: Source segments will be populated into offline downloaded files instead of matched target segments.

Preserve Source Folder Structure

If the "Preserve Source Folder Structure" option is checked, the "inbox" folder of the downloaded TransKit will contain the same folder/file (as bilinguals) structure as per the source (ZIP) used to create the job.

Consolidate/Split Type

Consolidate All Files - Checked to consolidate all segments from different pages into one offline downloaded file, such as RTF, Xliff file.

File by File - Offline download file by file.

Split File per Word Count - Split files per word count. Minimum word count of Download Setting should be 1000.

Include Repeated Segments as Separate File

Allows you to decide including one separate file which contains repeated segments into offline kit or not.

NOT include fully leveraged file(s)

Checked to NOT include fully leveraged file(s) in offline kit. A file is fully leveraged meant all segments in the file are 100% exact match or ICE during job creation.

Include XML Node Context Information

Allow XML node information to be included with the XLIFF export for translation.

Secondary Target Files

 

Select Pages

Select the pages you wish to localize offline by clicking the checkboxes next to each file name.

You can click Check All or Clear All to select all or none of the files at once.

Support Files

 

Select Support Files

Select the Support Files you wish to download by clicking the checkboxes next to each glossary.

The date next to the file can help you determine if the glossary has been updated since the last time you downloaded it.

Buttons

 

Cancel

Cancels the download.

Download

Downloads a .zip file using the selected settings.
