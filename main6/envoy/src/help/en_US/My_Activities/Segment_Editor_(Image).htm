Target Image

Match Results

Lists any fuzzy matches found in the TM for this segment.  Each match is listed with a percentage that indicates how close the matched source material is to the actual source.

Segment Versions

The segment as it appeared in each of the previous activities.

Save

Saves your work on this segment.

 

Keyboard equivalent: Ctrl+S

Previous Segment

Moves to the previous segment to translate.

You will be prompted to save the segment you are on.

 

Keyboard equivalent: Ctrl+PageUp

Next Segment

Moves to the next segment to translate.

You will be prompted to save the segment you are on.

 

Keyboard equivalent: Ctrl+PageDown

Revert

Reverts to the last saved version of this segment.

Details

Contains troubleshooting information about a segment that may assist GlobalSight Customer Support.

Close

Closes this window.

You will be prompted to save the segment you are on.

Keyboard Equivalents

Use the following keyboard equivalents for faster Segment Editor navigation:

Ctrl+S: Save segment

Ctrl+PageUp: Previous Segment

Ctrl+PageDown: Next Segment

Ctrl+B: Bold

Ctrl+I: Italic

Ctrl+U: Underline

Ctrl+LeftArrow: Word left

Ctrl+RightArrow: Word right

Shift+Ctrl+LeftArrow: Select word left

Shift+Ctrl+FightArrow: Select word right
