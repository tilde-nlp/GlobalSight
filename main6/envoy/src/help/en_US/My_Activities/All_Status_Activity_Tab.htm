All Status Activities

See also Available, In Progress,Finished, and Rejected.

 

Summary:

All status activities list here if you are the activity assignee or one project manager of the job.

  • Skipped activities do not appear in this list

Search

 

Status:

Select an activity status (Available, In Progress, Finished, Rejected,All Status) to filter the jobs displayed in the search results.

Advanced.../Simple...

Click on Advanced.../Simple... to change search mode.
In Advanced search mode, you can search activities by Acceptance Date and/or Estimated Completion Date.

Table

Displays all status activities.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Check the check box before each activity to select it or check the check box in table header to select all status activities.  Click on the job name to view the activity details.

Inputs

 

Job Id:

Select =,> or < and type job id to filter activities equal to, greater than or less than specified value.

Job Name

Filter activities by specified text in job name text filed.

Activity

Filter activities by specified text in activity text filed.

Assignees

Filter activities by specified text in assignees text filed.

Source Locale

Filter activities by selected source locale.

Target Locale

Filter activities by selected target locale.

Buttons

 

Search

Searches based on the criteria entered.

Detailed Word Count...

Select a single activities to view detailed word counts. This button is disabled when multipe activities are selected. (see Detailed Word Counts)

Export

Export target files for selected activities.you may need relevant permissions to enable this feature.

Offline Upload

Upload offline transiation kit without go to work offline page.
