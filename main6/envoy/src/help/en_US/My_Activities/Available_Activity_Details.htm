Available Activity Details

Summary:

View information about an activity and decide to accept or reject.

Tabs

 

Details

Refresh the detail page

 

Comments

View/Create/Edit comments for this activity (see Comments)

 

Details

 

Job Name:

Name of the job, may be automatically generated by the system.

Job ID:

Unique job ID number.

Activity:

Describes the type work to be performed.

Examples: Translation, Edit, Review

Project:

Name of the project.

Example: World Services R&D

Project Manager:

GlobalSight manager of the above project.

Source Word Count:

List number of words in the activity

Priority:

A number from 1 to 5 indicating the urgency to localize this content.

Source Locale:

The source locale of the content in this activity.

Target Locale:

The target locale for this activity.

Accept By:

The activity needs to be accepted by the listed date.

Time to Complete:

Amount of time allocated to complete this activity.

Overdue:

Indicates whether the Activity has not been completed by the Due Date indicated.

Status:

Indicates whether the activity is still available, has already been accepted, or rejected.

(Open/Closed Segment comment links)

Shows open and closed segment comments (see Segment Comments)

Primary Target Files

 

Target Page Name Search:

Filters pages whose page name contains the text in the adjacent blank text field.

(page name)

Click on a page name to see the content to localize in a read-only mode.

Word Count

Lists the number of  words for each page in this activity.

This icon next to a file name indicates that it is an extracted file.

This icon next to a file name indicates an un-extracted file.

Buttons

 

Search

Secondary Target Files

 

(page name)

Click on a page name to view the content to localize.

Buttons

 

Accept

You agree to complete your activity for the listed pages by the due date.  After clicking accept, see Accepted Activity Details.

Reject

You are unable to complete the activity.  Once you reject an activity you cannot accept it at a later date.

Export...

Export files (see Export)

 

As a project manager, you are unable to accept or reject activities that have not been assigned to you.
