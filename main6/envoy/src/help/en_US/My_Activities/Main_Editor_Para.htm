Shortcut: Alt-[ or

Alt-Ctrl-Home

Opens the next segment, if a segment is not already open, with the current translation in the target field.


Shortcut: Alt-\ or

Alt-Ctrl-Num+

Saves and closes the current segment and opens the next.


Shortcut:
Ctrl-S or Alt-] or Alt-Ctrl-End

Saves and closes the current segment.

Shortcut: Esc

Closes the current segment without saving.

Checks the spelling of the current open segment.

Undo the most recent change made to the segment.

Redo the most recent undo.

x2

Shortcut: Ctrl-8

Make the selected text in an open segment subscript.

x2

Shortcut: Ctrl-9

Make the selected text in an open segment superscript.

Shortcut: Ctrl-B

Make the selected text in an open segment bold.

Shortcut: Ctrl-I

Make the selected text in an open segment italic.

Shortcut: Ctrl-U

Underline the selected text in an open segment.

Insert a line break at the current cursor position.  This line break is displayed as a placeholder tag.

Insert a non-breaking space at the current cursor position.  This line break is displayed as a placeholder tag.

Display all placeholder tags for this segment.  Click a placeholder tag to insert it at the current cursor position.

Displays the details of this segment, including segment ID, Format and Type.  Also displays the code represented by any placeholder tags for this segment.

Shortcut: Ctrl-I

Open the translation memory window.  Any TM matches for open segments are displayed in this window.

Use machine translation to translate the current open segment.  After machine translation, you may need to restore ptags to the segment.  See the [ptag...] button above.

Merge Segment

Shortcut: Alt-Ctrl-PageUp

Merges the current segment with the next segment.

Split Segment

Shortcut: Alt-Ctrl-PageDn

Splits a previously merged segment.

Close Editor

Closes the main editor.  Any unsaved changes in open segments are lost.

Unlock/Lock

Unlock: Unlocks all exact matches and allow them to be edited.

Lock: Re-locks unlocked exact matches.

If your Project Manager has not enabled the Unlock option, the Unlock/Lock button is not shown.

Auto-Propagate

Click to open Auto-Propagate dialog.

This is only availabe for edit role.

Filter (Dropdown)

Filter the segments by categories.

Options

Opens a dialog window that allows you to better manage trailing white-space in certain segment types.

Help

File Tree

On the left side of screen, displays the target files of selected task, and every file's translated percentage will be displayed too. Click the file name to show its segments on the right main area. Select files and click "Approve" button to approve untranslated segments.
