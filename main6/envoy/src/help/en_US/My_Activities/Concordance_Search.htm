Concordance Search

Summary:

Lists matching strings found in the TM.  Matching string results can be viewed in context via Corpus Results.

Search Type

Select the type of search to perform.  See below for a description of each search type.

Full Text Search

 

Source Locale Search Text

Enter a string in the source locale to find in the TM.  All words in the string are used to find matches.  For example, searching for the string 'Acme brand' might result in the following matches: "The Acme Brand was founded in 1970." and "Acme is a brand recognized in more than 74 countries.".

 

It is possible to use an asterix ("*") as a wildcard

character. For example, corp* would match both "corp" and "corporation".  Warning: using only "*" in a search may result in an OutOfMemory exception.

Translation Memory

Select TMs in which to search.  Depending on your system configuration, only the TMs used in the project for this activity are listed, or all TMs are listed.

Exact/Fuzzy Match Search

 

Source Locale Search Text

Enter a string in the source locale to find in the TMs.  The entire string is used to find matches in the TM, and is compared to entire segments.

Translation Memory

Select TMs in which to search.  Depending on your system configuration, only the TMs used in the project for this activity are listed, or all TMs are listed.

TM Profile

Select the TM Profile to use for executing the search.  The TM Profile will specify the fuzzy match penalties for white-space, case, code, type and multiple exact matches.

Fuzzy Threshold

Select the fuzzy threshold to use for the string matching.  Only matches above this threshold are displayed.

Leverage From

Specify the target language for search results.

Document Name Search

 

Corpus Document Name

Enter a string to use for searching for specific Corpus documents.  This string will be matched to any part of the Corpus document name or path.  For example, if 'main' is entered, the following files might be found: en_US/main/index.html; en_US/main_nav.html

 

Results from this search are displayed with three options:

View Native Format - Opens the document.  You must have an application capable of reading this format.

View Text - View the text of the document, without opening it in its native format.

Remove - Remove the document from the Corpus TM.

Buttons

 

Search

Initiates the concordance search.

Search Results Table

 

Corpus Context

Click this icon to search for the match result in in the Corpus repository (see Corpus Results).

Percentage Match

Lists the percentage of the fuzzy match retrieved (Exact/Fuzzy Match Search Only).

Source Locale

Lists the matching source segment.  The string used for the search is highlighted.

Target Locale

Lists the corresponding target segment.
