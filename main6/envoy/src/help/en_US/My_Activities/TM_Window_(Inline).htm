TM Window (Inline)

Summary:

Displays TM matches for the current segment.

Source:

Shows the source for the segment.

Target:

Shows translation memory matches found.  Each match has a percentage score.  Moving the mouse over the match score, a tooltip appears displaying more information about the match.

Right click on the match to show options for copying the text into the editor or to perform a corpus search on this match.

Terminology:

Displays all terminology matches.  Right click on the match to show options for copying the text into the editor.

Buttons

 

Actions

Browse Termbase: Browse the entire termbase.

Re-leverage: Leverages the current segment against all approved TM matches as well as in-progress matches and the most current termbase.

Close

Closes the TM results window.
