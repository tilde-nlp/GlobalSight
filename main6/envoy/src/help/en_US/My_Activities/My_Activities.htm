My Activities

Summary:

Click on a tab to see the activities in that status.

Available

All activities available for you to accept appear under this tab.  If you are a project manager, all available activities for jobs you are managing appear under this tab.

  • Accepting an activity moves it to the In Progress tab.

  • Rejecting an activity moves it to the Rejected tab.

In Progress

All activities you have accepted appear under this tab.  If you are a project manager, all in progress activities for jobs you are managing appear under this tab.

  • Finishing an activity moves it to the Finished tab.

  • Un-Accepting an activity moves it to the Rejected tab.

Finished

All activities you have finished appear under this tab.  If you are the project manager, finished activities for jobs you are managing appear under this tab.

  • No actions are possible from this tab.

  • Once the localized content for an activity has been exported by the GlobalSight project manager the activity is removed from this list.

Rejected

All activities you have rejected or un-accepted appear under this tab, until these activities are accepted by another localization participant.  If you are the project manager, all rejected activities for jobs you are managing appear under this tab.

  • No actions are possible from this status.

  • Once the activity has been accepted by another user the activity is removed from this list.
