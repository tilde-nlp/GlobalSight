Unextracted File

Extraction is the process of separating the localizable text from non-localizable code.  However, not all data formats types can be extracted by GlobalSight (Supported Data Formats).  Localization of these data formats can still take advantage of GlobalSight's workflow engine by being imported as an unextracted file.

 

 This icon next to a file name indicates an un-extracted file.

 

 In contrast, this icon next to a file name indicates that it is an extracted file.
