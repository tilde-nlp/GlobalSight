Locale Manager

The Locale Manager group in GlobalSight is a user group (access level) that has special privileges on a per-locale basis. A user who is a Locale Manager has privileges that are between a Localization Participant and a Project Manager.

 

A Locale Manager can:

  1. Function as a Localization Participant (Translator)

  2. Add and Delete snippets to specific target locales to which they have been assigned.

  3. Publish translated pages to specific target locales to which they have been assigned.
