Extracted File

Extraction is the process of separating the localizable text from non-localizable code.  Many different file types can be extracted by GlobalSight, see Supported Data Formats.  Online, extracted files can be edited in the main editor.  Offline, extracted files can be identified because they will contain segments and each segment will have an ID number.

 

This icon next to a file name indicates that it is an extracted file.

 

In contrast, this icon next to a file name indicates an un-extracted file.
