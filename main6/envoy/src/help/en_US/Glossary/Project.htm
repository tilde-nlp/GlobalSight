Project

Each project consists of a name, description, project manager, termbase and assigned users.  This project manager is responsible for all jobs assigned to the project.  A job is assigned to a project based on the localization profile that creates it.  Each localization profile is assigned to one project.  The users assigned to the project can be modified by the project's project manager.

 

Projects are a good way to separate different globalization efforts within the same company.
