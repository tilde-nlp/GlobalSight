Content Management Overview (Optional)

All features of the content management module are detailed extensively in the Help specifically written for this application.  You can access this help via the content management module if you have access to the module itself.  Select the Content Manager link from the main menu and when the content management module appears, click Help.

 

To translate a content management resource, select the resource and click the translate tab/button.  Select the appropriate GlobalSight Host from the drop down, then select the media type.  Select the File profile to use for this resource listed under the selected media type.  Click Localize to begin localization of the resource with GlobalSight.
