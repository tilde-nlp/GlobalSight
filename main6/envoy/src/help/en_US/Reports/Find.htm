Find

Summary:

Find text within a report.  Optional

Search in

Specify an area of the report to search.

Column

Select a column from the report.

qualifier

Select a qualifier from the list to narrow the search criteria.

empty text box

Enter the text to search for.

Buttons

 

Search

Searches for the specified text.

Clear Search

Clears the search criteria and resets to the default options.

Cancel

Cancels the search.

Match case

Matches case with the search criteria.

Whole word only

Matches the search criteria only as a complete word.

More

Adds an additional row of fields to further narrow the search.

Fewer

Removes a row of fields to broaden the search.
