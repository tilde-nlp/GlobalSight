Vendor - Permissions

Summary

Add/Remove permission group(s) for the vendor.

Left Column (Available)

Lists all available permission groups.  Select a permission group by clicking on one, shift and ctrl can be used in combination with a click to select multiple permission groups.

Right Column

(Added)

Lists all permission groups added to the vendor.  Select a permission group by clicking on one, shift and ctrl can be used in combination with a click to select multiple permission groups.

Buttons

 

>>

Add selected permission group(s) to the vendor.

<<

Remove selected permission group(s) from the vendor.

Buttons (New)

 

Cancel

Cancel the creation of the Vendor.

Previous

Returns to Field Level Access.

Save

Save new vendor.

Buttons (Edit)

 

Cancel

Cancels changes made to vendor's permission group(s)

Done

Holds current changes.  Returns to Edit Vendor- Basic Information.

Note: Save must be used on the Edit Vendor- Basic Information screen to save current changes made to permission group(s).
