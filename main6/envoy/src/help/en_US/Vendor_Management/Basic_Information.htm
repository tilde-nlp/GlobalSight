Vendor - Basic Information

Summary:

Create or modify a vendor in GlobalSight

Is this vendor an existing GlobalSight user?

Yes, Username: If the vendor has already been added to the system, select the username from the dropdown.

No: If the vendor has not been added to the system, select this option.

GlobalSight Access:

Allowed: The vendor is granted (or already has) an GlobalSight login.

Not Allowed: The vendor is not granted an GlobalSight login.

Status:

Pending: The vendor has not yet been approved for work.

Approved: The vendor is approved for work.

Rejected: The vendor is not approved for work.

On Hold: The vendor is not currently approved for work.

Vendor Id:

Enter a Identifier for this vendor.  If you do not provide one, an ID will be automatically be generated.

*Username:

User login.  This value cannot be edited after the object has been saved.  The login may only contain letters, numbers, the underscore "_", the hyphen "-", the dot '.' or the at symbol '@'.  

*Password:

No restrictions on length or content.

*Repeat Password:

Re-enter the password.

First Name:

For display purposes.

Last Name:

For display purposes.

Alias:

For display purposes.

Title:

For display purposes.

Company Name:

Select a company from the dropdown list, or enter the name of a new company in the text box provided.

Type:

External: Vendor is external to the company.

Internal: Vendor is internal to the company.

Countries of Citizenship:

For display purposes.

Date of Birth:

For display purposes.

Notes:

These notes are only visible from the vendor management console.

 

*These options only appears if a vendor does not yet have an GlobalSight username, access is allowed, and status is approved.
