Custom Form Designer

Summary:

Create a custom form to describe vendors.

Sections

 

Table

Lists custom sections that have been added to the form.  Select a section by clicking the radio button next to the name of the form.

Section Name:

Enter the name of the new section to add.

Buttons

 

Add

Add a new section to the custom form.  You must first enter a name under Section Name.

Remove

Remove the selected section.

Fields In Section

 

Table

Lists all the fields in the selected section.  Select a field by clicking the radio button next to the name of the field.

Label:

Enter a name for this field.

Type:

Text: The field appears with a text area for typing comments.

Checkbox: The field appears with a checkbox.

Buttons

 

Add

Adds a new field to the section.  You must first enter a name under Label.  

Remove

Removes the selected field.

Buttons (Page)

 

Remove

Remove the custom form.

Preview

Preview the custom form.

Save

Save the custom form.

Done

Saves the custom form and returns to the Vendors main screen.
