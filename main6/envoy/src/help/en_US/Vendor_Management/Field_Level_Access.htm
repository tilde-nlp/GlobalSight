Vendor - Field Level Access

Summary:

Set access to the fields describing this vendor.

Table

Displays all fields describing this vendor.  Select the radio button next to the field to determine how users who do not belong to this vendor's projects will be able to access this vendors's information.

Access

shared: this field is editable by anyone that can view the vendor.

locked: this field can only be viewed, not edited.

hidden: the contents of this field are hidden and cannot be viewed or edited.

Buttons (New)

 

Cancel

Discard new vendor.

Previous

Return to Projects.  If custom forms have been added, the previous button navigates to these instead of projects.

Next

Add permission groups (see Permissions)

Buttons (Edit)

 

Cancel

Discard changes and return to Basic Information.

Done

Save changes and return to Basic Information.
