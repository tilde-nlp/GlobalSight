Search Vendors

Summary:

Filter vendors displayed on the Vendors Users Screen.

Name:

name dropdown: Select the type of name to filter on.

qualifier: Select the qualifier for the name to filter on.

blank text box: Specify the name to filter on.

Company Name:

qualifier: Select the qualifier for the name to filter on.

blank text box: Specify the name to filter on.

Source Locale:

Filters vendors qualified for an activity in this source locale.

Target Locale:

Filters vendors qualified for an activity in this target locale.

Rate:

rate type: Select the rate type to filter on.

qualifier: Select the qualifier for the name to filter on.

blank text box: Specify the rate value to filter on.

Activity Type:

Filters the vendors displayed by qualification in the activity type selected.

Match Case:

Checked: Matches capitalization for names specified in search criteria.

Unchecked: Ignores capitalization for names specified in search criteria.

Keywords:

Filters the vendors displayed by words in the custom vendor screen.

Buttons

 

Cancel

Returns to the Vendor Main Screen.

Search

Filters the vendor list based on the selected criteria.
