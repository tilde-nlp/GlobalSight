Email Notification

Summary:

Assign users to a project.  Only those users assigned to a project are qualified to complete activities in that project.

Enable Email Notification

Checked: The user receives the email notifications selected below.

Unchecked: The user receives no email notifications.

Table

Notifications are displayed in a table.

Available:

Lists all notifications that the user may receive.  Select a notification by clicking the name of the notification.

Send me the following emails:

Displays all notifications that the user will receive.  Select a notification by clicking the name of the notification.

Email Notifications

Delayed re-import: You receive an email when a file in an activity you have accepted is being re-imported with updates.

Export completion report: You receive an email when a file is successfully exported.

Export for update: You receive an email when a file with adds and deletes in an activity you have accepted is being exported to be updated.

Initial import failure: You receive an email when a file you have selected for import experiences an import failure.

Newly assigned activity: You receive an email when an activity becomes available for you to accept.

Saving segments failure: You receive an email when a segment fails to save.

Successful upload: You receive an email when you successfully upload a file.

Workflow cancellation: You receive an email when a workflow in which you have accepted an activity is cancelled.

Buttons

 

>>

Adds the selected notification to "Send me the following emails".

<<

Removes the selected notification from "Send me the following emails".

Cancel

Returns to the account options screen without saving changes.

Done

Saves changes and returns to the account options screen.

Edit

Edits email template content.

Save

Saves changed email template content.

Reset

Resets email template content to the original email template.
