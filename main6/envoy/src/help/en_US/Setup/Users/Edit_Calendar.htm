Create/Edit a User Calendar

Summary:

Edit the user's working calendar by choosing a base calendar and adding or removing events such as vacation days, and modifying working hours as necessary. Working days and hours are used to calculate estimated "time to completion" for activities the user has accepted. Optional

Base Calendar

Select a calendar on which to base the user's workday schedule.

Time Zone

Select the user's time zone.

Month/Year

Select the month and year to display on the calendar shown.

Calendar Table

The selected calendar shows working and non-working, as well as days for which events have been scheduled.  Click a day to View events on that day.

Working Hours Per Day

 

Working Day:

Select the day for displaying and editing the user's working hours.

From: / To:

Enter the user's actual working hours by modifying the From and To fields appropriately.  For example, if the user works from 8am to 12pm, and 1pm to 5pm, enter 8am in the first From field, and 12pm in corresponding To field.  Then enter 1pm in the next To field, and 5pm in the corresponding From field.
