Roles

Summary:

Set up a user's default qualifications.

Table

Displays all locale pairs in which the user is qualified for one or more activities as default.
When user is added into a project in sub company, the default qualifications of user which is fit for the locale pairs and activities in sub company can be set up automatically in sub company.
Note: This operation is only used for users who are in the super company.

Buttons

 

Cancel

Discard changes and return to User - Basic Information.

New...

Qualify the user for activities in a new locale pair as default (see New/Edit Roles).

Edit...

Change a user's default qualifications in the selected locale pair (see New/Edit Roles).

Done

Save changes and return to User - Basic Information.
