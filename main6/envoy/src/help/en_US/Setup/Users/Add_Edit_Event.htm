Add/Edit Event

Summary:

Add or Edit an event for a user. Optional

Subject

Display name for the event.

Date:

Date the event is to take place.

Time:

All Day - Event lasts all day.
Start/End Time - Event has a specific start and end time.

End Date:

One Day Only - Event occurs only once.
Until - Select the final day month and year of the event.

 

 

Buttons

 

Cancel

Cancel new/changed event and returns to Events and Activities for the day.

Done

Save the new/changed events.
