Upload Support Files

Summary:

Add or update a support file.

Source Locale

Select the source locale for which this support file will be available.

Target Locale

Select the target locale for which this support file will be available.

Select file to upload:

Use the Browse... button to select the support file, and the Upload button to upload it to GlobalSight.

Buttons

 

Browse...

Locate the support file on your PC to upload.

Upload

Uploads the selected file to GlobalSight.

Close

After upload is complete, "Upload Successful" appears on the screen.  Click Close to go back to the Support Files Main Screen.
