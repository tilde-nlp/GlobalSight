Support Files Main Screen

Summary:

Add or remove a support file, to the system.  Support files can be locale pair specific and are available to users during the localization process

Table

Displays all support files that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a support file by clicking the check box next to the support file.  Click a filename to open that support file, right-click to download.

Buttons

 

Remove

Remove the selected (checked) support files.

Upload...

Add or update an existing support files.

Return to Activity

If you access this page from an activity you can return to that activity by clicking this button.
