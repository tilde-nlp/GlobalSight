Assigning Holidays

Summary

Assign system holidays to a system calendar.  Optional

Left Column

Lists all system holidays.  Select a holiday by clicking it, shift and ctrl can be used in combination with a click to select multiple holidays.

Right Column

Lists all system holidays assigned to this calendar.  Select a holiday by clicking it, shift and ctrl can be used in combination with a click to select multiple holidays.

Buttons

 

>>

Add selected system holidays to this system calendar.

<<

Remove selected holidays from this system calendar.

Cancel

Cancel the creation or editing of the holidays for this system calendar without saving changes.

Save

Saves changes made to holidays in this system calendar.

Create Holiday

Retains holiday assignments and opens the Create System Holiday screen.
