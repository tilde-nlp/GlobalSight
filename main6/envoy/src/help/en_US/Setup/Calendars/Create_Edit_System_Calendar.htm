Create/Edit System Calendar

Summary

Create or Edit a system calendar.  This includes defining working hours for each working day, and defining holidays. Optional

Name

Enter a unique name for the system calendar.

Business Hours per Day

Select the number of business hours in a typical working day.

Time Zone

Select the timezone for the system calendar.

Holidays

Click to define system holidays and/or assign holidays to this calendar (see System Holidays).

Working Hours Per Day

 

Working Day

Select the day for displaying and editing the system calendar's working hours.

From: / To:

Enter the actual working hours for the system calendar by modifying the From and To fields appropriately.  For example, if the working hours are normally from 8am to 12pm, and 1pm to 5pm, enter 8am in the first From field, and 12pm in corresponding To field.  Then enter 1pm in the next To field, and 5pm in the corresponding From field.

Calendar Table

Displays the current calendar.  Days in displayed in red are non-working days (includes weekends and holidays).  Days displayed in black are working days. Select the month and year of the calendar to display.

Buttons

 

Cancel

Cancels the creation or editing of the system calendar without saving changes.

Apply

Update the display of the calendar using the Working Hours Per Day definitions.  The calendar will not display new definitions until Apply is used.

Done

Save the calendar information and return to the previous screen.
