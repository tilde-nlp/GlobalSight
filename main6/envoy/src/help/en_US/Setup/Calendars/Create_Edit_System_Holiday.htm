Create/Edit System Holiday

Summary

Create or edit a system holiday. Optional

Name

Enter a display name for the system holiday.

Description

Enter an optional description for the system holiday.

NonWorking Day

Select the date of the holiday.  

Month/Day: Select the day and month for the holiday .

Descriptive: Specify a particular day in a month (e.g. the third Thursday in November).

End Date

No end date:the holiday occurs every year.

Only: Specify the year in which the holiday occurs.

Buttons

 

Cancel

Cancels the creation or modification of the holiday without saving the changes.

Save

Save the creation or modification of the system holiday.
