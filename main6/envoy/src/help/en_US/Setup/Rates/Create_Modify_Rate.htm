Create/Edit Rates

Summary:

Determine a per word costs for an activity in a locale pair. Optional

Name:

For display in GlobalSight.  This name cannot be edited after the rate has been saved.  The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Activity Type:

Select the activity type to which this rate applies.

Locale Pair:

Select the locale pair to which this rate applies.

Currency:

Select the currency for the below rates.

Rate Type:

Select the currency for the below rates.

Rate Types

Word Count(by %)

Word Count rates specify the cost per word for localization.

Exact/Fuzzy Match Percentages

Enter the rates for the exact match percentage and the fuzzy match percentages in the above currency.

No Match:

Enter the rate for no matches in the above currency.

No Match Repetitions:

Enter the rate for no match repetitions in the above currency.

Context Matches

Enter the rate for exact matches from page TM in the above currency.

Decimal Digits:

To designate the decimal digits when calculate the rate for "Word Count by %" type.

Fixed

Fixed rates specify the cost per Activity for localization. For example, if you assign a Fixed rate of $20.00 to the English to Spanish Translation Activity, then each time this Activity (or Role) is completed in a workflow, $20 will be added to the Actual Cost.

Rate:

Enter the cost per Activity.

Page

Page rates specify the cost per page for localization. For jobs that have page-based rates, the Project Manager can manually enter the number of pages in the job from the Job Details screen. The Localization Participant can also enter in the number of pages in a job from the Activity Details screen, if it differs in the localized document.

Rate:  

Enter the cost per Page.

Hourly

Hourly rates specify the cost per hour for localization. On the Activity Details screen, each Localization Participant enters in the hours they spent on an Activity.

Rate:

Enter the cost per Hour.

Buttons

 

Calculate

Calulate the rate based on the "Baesd Rate(No Match)" for "Word Count by %" type.

Cancel

Cancels rate creation/edit.

Save

Save the new/changed rate.
