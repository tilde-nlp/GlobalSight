Import Schedules

Summary:

Import entries from an exported schedule in the form of an delimited text file.

Select a file to import:

Use the Browse... button to select a CSV project from your local machine or shared drive.

Import file format:

CSV: The file to import must be a delimited text file.

File encoding:

Select the file encoding used when saving the file.  English and western-European languages typically use Windows-1252.

Column delimiter:

Select the delimiter used when saving this file.

First line is column header, skip during import:

Checked: The first line of the import is ignored and used only as a header.

Unchecked: The first line of the import is treated as schedule data.

Buttons

 

Cancel

Cancels the schedule import.

Next

Moves to the next screen of the TM import (Verify CSV Columns).
