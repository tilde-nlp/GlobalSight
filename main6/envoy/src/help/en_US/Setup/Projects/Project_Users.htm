Project Users

Summary:

Assign users to a project.  Only those users assigned to a project are qualified to complete activities in that project.

Table

Users assigned to the project are displayed in a table.  Use the Previous and Next buttons to page through all users assigned to the project.

Available:

Lists all users available to be added to this project. Select a user by clicking the user.

Added:

Displays all users added to the project  Select a user by clicking the user.

Buttons (New)

 

>>

Moves the selected user from "Available" to "Added".

<<

Moves the selected user from "Added" to "Available".

Cancel

Returns to the projects main screen without saving changes.

Previous

Returns to create/edit projects screen.

Save

Saves and returns to the projects main screen.

Buttons (Edit)

 

>>

Moves the selected user from "Available" to "Added".

<<

Moves the selected user from "Added" to "Available".

Cancel

Returns to the project create/edit screen without saving changes.

Done

Saves changes and returns to the project create/edit screen.
