Export Schedule

Summary:

Export all user schedules from a project.  The exported schedules can later imported by GlobalSight or by another program that utilizes translation memory.

Select users whose schedules should be exported:

All Users: Exports all user schedules from the project.
Only user: Exports only the user's schedule selected from the dropdown.

Export format:

CSV: Exports are in a delimited text file.

File Encoding:

Event names may contain special characters.  Select the encoding that will include any special characters and work with the application you intend to import the schedule into.

Buttons

 

Cancel

Cancels the project schedule export

Next

Moves to the next screen of the project schedule export (Output Options)
