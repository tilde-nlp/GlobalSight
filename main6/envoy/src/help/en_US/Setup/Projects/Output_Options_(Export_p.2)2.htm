Output Options (Export p.2)

Summary:

Specify options for user schedule export.

Write header line containing column names:

Checked: Column names are written at the top of the exported file.
Unchecked: The exported file begins immediately with schedule data.

Column delimiter:

The selected value is used to separate the schedule data in the exported file.

Date format:

Select the date format to be used in the exported file.

Buttons

 

Cancel

Cancels the schedule export.

Previous

Returns to the previous screen.

Export...

Proceeds with schedule export using the selected options.
