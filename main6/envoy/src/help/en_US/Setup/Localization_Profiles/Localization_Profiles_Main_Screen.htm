Localization Profiles Main Screen

Summary:

Describe how content should be localized.  Includes a workflow template for each target locale, as well as size and dispatch criteria for jobs created by this localization profile.

Table

Displays all localization profiles that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a localization profile by clicking the checkbox next to the localization profile.Click the "name" link to edit localization profile. Input string for "name", "Translation Memory Profiles", "company" ,"Project" and click "enter" to filter data.

Buttons

 

Remove

Remove the selected localization profile.

Details

Show source/target locales, dispatch and size criteria for the selected localization profile (see Details)

Duplicate...

Duplicate the selected localization profile.  The localization profile must have only one workflow attached.  Duplicating the localization profile results in the duplication of the attached workflow as well (see Duplicate Localization Profile)

New...

Add a new localization profile to the system (see Basic Information).

Report...

Download a report for localization profile(s).
