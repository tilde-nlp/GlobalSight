Details

Summary:

View details for the selected L10N profile.  (For more details on a L10N Profile field see Localization Profile Basic Information)

Name:

Displayed name for the L10N profile.

Workflow Dispatch:

Will Show if the Workflow has automatic or manual dispatch.

 

Automatic – Once an import is complete, it moves from Pending to In Progress, and the first participant is notified to begin.

Manual – Once an import is complete, the job is placed in Ready.  From Ready, the project manager can decide whether to discard the job or move it to In Progress.

Source Locale:

This is the selected locale of source content.

Target Locale(s):

Shows target locale(s).
