Currency Main Screen

Summary:

Specify the currency conversion rate from various currencies to the system currency. Optional

Table

Displays all currencies that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a currency by clicking anywhere in the currency's row.

Buttons

 

Edit

Modify the selected currency's exchange rate (see Create/Edit Currency).

New...

Add a new currency conversion rate to the system.

If you are using the automatic currency conversion feature, nearly every necessary conversion is already added to the system.
