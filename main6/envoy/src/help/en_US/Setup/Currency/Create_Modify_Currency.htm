Create/Edit Currency

Summary:

Determine the exchange rate for a given currency. Optional

System Default Currency:

Displays the System Default Currency.  This value is not editable, see System Parameters.

Display Currency:

Select the currency to enter a conversion rate for.  This value cannot be edited after the object has been saved.

Conversion Factor:

Enter the number of display currency you would get for 1 of the system default currency:

Example: System default currency is US dollar, display currency is Euro. If the exchange rate is 1 USD = 1.1 Euro, enter 1.1 in the conversion factor text area.

Buttons

 

Cancel

Cancels currency creation/edit.

Save

Saves the new/changed currency.
