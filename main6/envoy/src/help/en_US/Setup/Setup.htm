Menu Item

Description

 

Locale Pairs

Identify source-target locales for system-wide use.

Activity Types

Create/Edit activities for users to perform as part of the localization process.

Currency

Specify the currency conversion rate from various currencies to the system currency.

Rates

Determine the per word costs for activities in locale pairs.

Users

Create/Edit user information and access to the system.

Translation Memory

Create empty TMs in which to store translations.

Translation Memory Profiles

Establish parameters for how translation memory should be used within the system.

Machine Translation Profiles Profiles

Establish parameters for how machine translation should be used within the system.

Terminology

Maintain terminology databases to be used as references during the localization process.

Projects

Create/Edit projects to match content localization groups.

Workflows

Create/Edit workflows for routing translation jobs through the system.

Localization Profiles

Create/Edit profiles for completing localization work.

Support Files

Upload locale-pair specific glossaries for localization participants.

System Parameters

Change several system-wide parameters (incl. fuzzy matching threshold, web server admin email).

Logs Files

GlobalSight logs to assist in resolving system issues.

Operation Log

GlobalSight operation logs to assist in investigating system issues.
