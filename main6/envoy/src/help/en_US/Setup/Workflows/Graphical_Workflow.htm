Graphical Workflow

To create a workflow, you must have a Start Node, an End Node, one or more Activity Nodes, and arrows that connect all the nodes. Use the buttons at the top left of the Graphical Workflow canvas to create and connect the nodes.

 

To use the buttons, click the button you want to use, then click in the workspace. The selected workflow element will then appear in the workspace.

 

See sample workflows for examples: One Activity, Two Activities, Looping, and Looping with Finish Options.

 

Button Descriptions

 

Button

Name

Description

Activity Node Button

Creates an Activity Node in the workspace.

End Node Button

Create an End Node by clicking once in the workspace.

Conditional Node Button

Creates a Conditional Node in the workspace.

Arrow Connector Button

Connects two nodes in the workspace. Use it by clicking on one node, then drag to another node, then release.

Pointer Button

Turns the cursor into a pointer that lets you select workflow element to delete or to move around the screen.

Save

Save Button

Saves the workflow.

Cancel

Cancel Button

Cleans all inputs and go back to workflows screen. Workflow creation is cancelled.

Previous

Previous Button

Goes back to the previous screen.

 

 

Workflow Element Descriptions

Element

Name

Description

Start Node

The Start Node is where the workflow starts. All workflows must have a Start Node.

End Node

The End Node is where the workflow ends. All workflows must have a End Node.

Activity Node

An Activity Node is where you assign a user or users to an Activity (Stage) in the workflow.

 

Right-click on the Activity Node and choose the Properties pop-up window to assign users to the activity and to configure other activity settings. Connect the Activity Nodes to other workflow elements using the Arrow Connector tool.  See Edit Activity in Workflow.

 

You can have an unlimited number of Activity Nodes in a workflow.

Conditional Node

A Conditional Node lets you provide several different paths for a workflow. By extending several different Connector Arrow from a Conditional Node, you allow the active workflow user to choose where the workflow goes next. See  Sample Workflows for examples.

Arrow Connector

The Arrow Connector connects the elements of a workflow. The Action box on the arrow is for visual clarity only, it does not appear to the workflow users.

 

Arrows in the default path are highlighted in BLUE color. For the arrows in BLUE color but temporarily not connected in the default path, they could be part of the default path if the workflow advances to their direction by the condition node's selection.
