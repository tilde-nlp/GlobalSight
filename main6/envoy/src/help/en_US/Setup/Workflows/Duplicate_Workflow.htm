Duplicate Workflow

Summary:

Copy the graphical element of the selected workflow with different locale pairs.  
Notes: Project, Translation Memory Profile, Use Translation Memory option and Workflow Dispatch will be the same in the duplicated localization profiles.  In duplicated workflows, Project Manager and Workflow Manager remain the same. The "leverage from" option is set to the target locale.  Activities are assigned by default to all qualified users, and any costing information is changed to "No Rate".  

Name:

Enter a name to be used as a prefix for the duplicated localization profiles and workflows.  Duplicated localization profiles and wofkflows will have the name entered, plus the new source and target locale separated by underscores.  Example: if "My Copies" was the name prefix chosen, and en_US -> de_DE, en_US -> fr_FR were the locale pairs chosen, two new workflows would be created named My Copies_en_US_de_DE and My Copies_en_US_fr_FR.

Project:

Select the project in which this duplicated workflow is used.  Only users assigned to this project may perform work for this workflow.

Source Locale:

Select the source locale for the duplicate localization profiles and workflows

Target Locale:

Select the target locales for the new duplicate localization profiles workflows.

Locale Pairs to Duplicate:

Use the >> button to generate the workflow locale pairs from the selected source locale and target locales.  Use the << button to remove workflow locale pairs from the list.

Buttons

 

Save

Creates new localization profiles and attaches new workflows with the same graphical elements as the selected workflow with the selected locale pairs, and returns to the previous screen.

Cancel

Cancels localization profile duplication and returns to the previous screen.
