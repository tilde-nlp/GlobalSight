Edit Activity in Workflow

Summary:

Set the properties for an activity in a workflow.

Activity Type:

Select the activity type for this step in the workflow.

System Action:

None: No additional actions are taken.

Create Secondary Target Files: exports the files at this activity within the workflow.  These files are attached to the workflow, allowing participants to continue using the workflow functionality of GlobalSight while working with the files outside the GlobalSight editing environment.  Creating a secondary target file can only be done once per workflow.

Export Target Files: exports all the files at this activity within the workflow (the same as if a user had used the export button to export all the files for the workflow).

Time to Accept:

Specify the time a participant has to accept the activity.  Workflow managers are notified if the participant does not accept the activity in the allotted time.  Enter the number of days next to "d", the number of hours next to "h" and the number of minutes next to "m".

Time to Complete:

Specify the time a participant has to complete the activity once it has been accepted. Enter the number of days next to "d", the number of hours next to "h" and the number of minutes next to "m".

Participant(s):

All Qualified Users: All qualified users from this workflow's project are notified when this activity is ready for acceptance.

The users that can complete within deadline:  Only qualified users from this workflow's project that are not scheduled to work when the activity is ready for acceptance are notified and permitted to accept the activity.

The users that can complete earliest:: Only those qualified users from this workflow's project that are scheduled to work the least for the duration of the activity are notified and permitted to accept the activity.

A selected user... Select a specific qualified user from this workflow's project to perform this activity.

 

Internal Costing Rate:

Select the rate to be used for calculating the internal costs for this activity. Optional

Billing Charge Rate:

Select the rate to be used for calculating the billing charges (external costs) for this activity. Optional

Buttons

 

Cancel

Cancels the activity modification.

OK

Saves the changes to the activity.
