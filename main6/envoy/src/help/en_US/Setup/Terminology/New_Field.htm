New Field at level

Summary:

Enter values for a field in a term.

Field Type

Select the appropriate field type from the list.  Some field types will have attributes, others have text.

Attribute Value

Select the attribute value from the list if the field type it.

Text Value

Enter text for the field if the field type permits it.  Use the standard word formatting tools to format the entry.  Use the link icons to add hyperlinks (see Add Link), cross-reference a term (see Term Link). or insert an image (see Insert Image).

Buttons

 

OK

Save changes to the new field.

Cancel

Cancel the creation or edit of the new field.
