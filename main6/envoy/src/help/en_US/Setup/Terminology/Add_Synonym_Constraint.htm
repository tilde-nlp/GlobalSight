Add Synonym Constraint

Summary

Add a field constraint to the input model.

Synonyms in language:

Displays the language of the term to which a synonym constraint is being added.

are:

Required: the field must be complete in order to save the termbase entry.

Optional: the field may be completed when saving the termbase entry.

can occur multiple times

Multiple synonyms can be listed with the termbase entry.

Buttons

 

Ok

Saves the synonym constraint and closes the window.

Cancel

Closes the window discarding any changes.
