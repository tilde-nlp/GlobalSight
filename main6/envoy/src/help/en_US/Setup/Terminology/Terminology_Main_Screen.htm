Terminology Main Screen

Summary

Maintain terminology databases to be used as references during the localization process.

Table

Displays all termbases that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a termbase by clicking the radio button next to the termbase.

Buttons

 

Statistics

Display the number of entries, as well as the number of terms in each language of the selected termbase.

Indexes

Index a termbase for better performance (See Termbase Indexes).

Remove

Remove the selected termbase.

Duplicate...

Create a copy of the selected termbase (see Create/Edit/Duplicate Termbase).

Edit...

Modify the selected termbase  (see Create/Edit/Duplicate Termbase).

New...

Creates a new, empty terminology database (see Create/Edit/Duplicate Termbase).

Browse...

View terms in the selected termbase, manager users can edit termbase entries.

Import...

Imports new terms into the selected termbase (see Import Termbase).

Export...

Exports terms from the selected termbase (see Export Termbase).

Maintenance...

Search and replace entries in the selected termbase (see Termbase Maintenance).

Input Models...

Define the basic structure of termbase entries (see Manage Input Models).

Users

Modify the users who can access this TB, only Administrator user can see this button.
