Set Import Options (Import p.3)

Summary

Select synchronization options

Add all concepts as new

All concepts in the file are added to the termbase.

Synchronize on concept id

Overwrite existing concepts: If two concepts have the same id, the existing concept is overwritten by the new concept. Extra language term data in system will be remained.

Merge new and existing concepts:  If two concepts have the same id, the new concept is merged with the existing concept.  

Discard new concepts: If two concepts have the same id, the new concept is discarded.

Synchronize on language

Language Dropdown: Select the language on which concepts should be synchronized.

Overwrite existing concepts: If two concepts have the same term in the selected language, the existing concept is overwritten by the new concept. Extra language term data in system will be remained.

Merge new and existing concepts:  If two concepts have the same term in the selected language, the new concept is merged with the existing concept.

Discard new concepts: If two concepts have the same term in the selected language, the new concept is discarded.

Unsynchronized Entries

This option works for the concepts that do not appear in the existing termbase when synchronizing on concept id or language. When select "Add all concepts as new", this option will be ignored.

Discard: New concepts (those not in the existing termbase) will be discarded.

Add: New concepts (those not in the existing termbase) will be added to the termbase.

Buttons

 

Cancel

Cancels the termbase import.

Previous

Returns to the previous screen.

Import

Proceeds with termbase import using the selected options.
