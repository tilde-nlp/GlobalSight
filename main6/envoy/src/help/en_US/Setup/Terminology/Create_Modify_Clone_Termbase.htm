Create/Edit/Duplicate Termbase

Summary:

Name and describe specific termbases.

Name

For display in GlobalSight.  The name may not contain the hyphen character "-", or percent symbol "%".  

Description

Optional, for display only.

 

Define Termbase Languages

Table

Define the languages used in the termbase.  Select a language by clicking the radio button next to the language.

Buttons

 

Remove

Remove the selected language from the termbase.

Add

Identify a new language for use in the termbase

Define Fields

Table

Define the fields that appear in the termbase.  Select a field by clicking the radio button next to the field.

Buttons

 

Remove

Remove the selected field.

Modify

Modify the selected field (see Create/Edit Field).

Add

Add a new field to the termbase (see Create/Edit Field).

Save New Termbase

Buttons

 

Cancel

Cancel termbase creation/edit/duplication.

Save

Save the new/changed/copied termbase.
Note: After creating a new termbase, you must select the languages to be indexed in order for the termbase to be used for leveraging (see Termbase Indexes)
