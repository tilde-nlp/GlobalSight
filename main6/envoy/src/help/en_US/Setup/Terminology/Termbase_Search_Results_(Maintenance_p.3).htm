Termbase Search Results (Maintenance p.3)

Summary

Displays search results.

Searched for:

Displays the text searched for.  "Highlight" marks the searched text in the results displayed below.

Replace with:

Enter text to replace the searched text.

smart replace

Checked: Case is ignored when doing the replacement.  The original case of the text to be replaced is maintained wherever possible.

Unchecked: The case of the "Replace with:" text is used.

Buttons

 

Check All

Selects all results on the current screen for replacement.

Clear All

Clears all results for replacement.

(1-10 of 55)

Indicates the total number of results and shows which results are currently being displayed.  Use the arrows to either side of the numbers to navigate through the results.

Cancel

Cancel termbase maintenance.

Previous

Return to Termbase Maintenance.

Replace

Replace the "Searched for:" text in the marked segments with the "Replace with:" text.

Back

Return to Termbase Maintenance (No Results)
