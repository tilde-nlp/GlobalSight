Import

Summary:

Imports new terms into a termbase by uploading a formatted text file.

Browse...

locate the termbase text file on your PC to import.

File Options

 

Import file format

Select the format of the termbase file:

XML, TBX, Excel, TRADOS MultiTerm® XML, CSV

(Note: All data in CSV files must use a single encoding, e.g., UTF-8).

File encoding

CSV only: select the encoding of the file.

First line is column header

CSV only: if the first line of the delimited text file is a header, check the box.

Column delimiter

CSV only: Identify the delimiter used in the text file.

Buttons

 

Cancel

Cancels the termbase import.

Next

Moves to the next screen of the termbase export.
