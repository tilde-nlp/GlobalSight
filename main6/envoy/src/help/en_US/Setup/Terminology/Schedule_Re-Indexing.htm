Schedule Re-Indexing

Summary:

Specify the times at which the termbase should be re-indexed.

Runs At:

Displays the time at which the re-indexing will be run.

Select the tab that corresponds to the time interval at which you wish the re-indexing to be run.

Daily

 

Start time:

Specify the time of day to run the re-indexing.  Select one of the options below:

Every day: Runs the re-indexing at the specified time every day.

Weekdays: Runs the re-indexing at the specified time each week day.

Every ... day(s): Runs the re-indexing at the specified time every selected number of days.

Weekly

 

Start time:

Specify the time of day to run the re-indexing.  Also specify the information below:

Every ... week(s): Runs the re-indexing at the specified time every selected number of weeks on the day(s) specified below.

Select day(s) of the week: Select the day or days of the week you wish the re-indexing to be run.

 

Monthly

 

Start time:

Specify the time of day to run the re-indexing.  Select one of the options below and specify the months in which to run the re-indexing.

Day: Select the day of the month to run the re-indexing.

The: Describe the day of the month to run the re-indexing.

Once

 

Start time:

Specify the time of day to run the re-indexing.  Select the date on which the re-indexing should be run.  Click the calendar icon to view dates for selection.  Double click a date on the calendar to select it.

Buttons

 

Previous

Returns to the previous screen.

Unschedule

Unschedules any scheduled re-indexing.

Schedule

Schedules the indexing based on the criteria selected in the current tab.
