Add Link

Summary:

Insert or edit a hyperlink for a Termbase field.

URL

Enter a URL for the link.

Upload

Alternately, Browse to a file to link to.

Target

<Not Set>: Opens the link in the default window.

New Window (_blank): Opens the link in a new window.

Title

Specify a title for a new window (if applicable).

Buttons

 

OK

Save changes to the new link.

Cancel

Cancel changes to the link.

Browse

Browse to a file to be linked.

Send it to the Server

Send the browsed file to the server.

Browse Server

Browse for a file to link on the server.
