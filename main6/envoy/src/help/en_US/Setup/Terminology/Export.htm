Export

Summary:

Export terms from a termbase.

Select entries to export:

Entire termbase: exports all entries in the termbase sorted by entry id.

By language: only entries with terms in that language are exported, sorted by the language's sort order.

Filter entries by:

Use the criteria listed in this category to filter the entries to be exported from the Termbase.

Export format:

GlobalSight Format: Exports an XML file.

TRADOS MultiTerm® iX: Exports a Trados compatible XML file.

HTML: Exports an HTML file with Termbase entries, suitable for printing.

File encoding

The default option of UTF-8 is correct.

Buttons

 

Cancel

Cancels the termbase export.

Next

Moves to the next screen of the termbase export (Output Options).
