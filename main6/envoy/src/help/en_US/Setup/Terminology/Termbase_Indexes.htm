Termbase Indexes

Summary:

Index termbase fields to enable termbase leveraging.

Table

Displays all languages that have been added to the termbase.  Select concept-level or languages by clicking the check box next to the name.

Buttons

 

Previous

Returns to the previous screen.

Refresh

Refreshes the indexing results for the termbase.

Save

Saves the indexing.

Re-Index

Indexes the selected termbase languages and/or concepts.  

Schedule Re-Indexing

Specify the times at which the termbase should be re-indexed (see Schedule Re-Indexing).
