Termbase Maintenance

Summary

Search and replace in a termbase.

Search for:

Enter text on which to search.

Match case

Checked: Search is case sensitive.

Unchecked: Search is not case sensitive.

Whole word only

Checked: Exact search with the input text.

Unchecked: Fuzzy search with the input text.

Select one of the three options below for the search.

on the concept level

Select this option to confine the search to fields at the concept level (Definition, Source, Note). If select the blank option, search in all concept types.

on language level in language

Select this option to confine the search to fields at the language (Source, Note).  Select a language in which to perform the search. If select the blank option, search in all language level types.

on term level in language

Select this option to confine the search to fields at the term level (Definition, Context, Example, Source, Note).  Select a language in which to perform the search. If select the blank option, search in all term level types.

Buttons

 

Previous

Cancel the search and replace, return to Terminology Main Screen.

Browse Termbase

Browse this termbase.

Search

Perform the specified search.

Replace with:

Enter text to replace text found in the search.
