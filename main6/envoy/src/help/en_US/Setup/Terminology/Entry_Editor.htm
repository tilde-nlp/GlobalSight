Entry Editor

Summary:

Edit a termbase entry.

Help

View help on the termbase editor.

Close the Entry Editor window.

Editor Menu

 

Validate Entry

Check to see if this term appears more than once in the termbase.

Approve Entry

Change the status of the entry to proposed, reviewed, or approved.  The default status is approved.

Save Entry

Saves all changes made to the entry.

Close Editor

Closes the editor.

Delete Entry

Deletes the entire termbase entry.

Add Language

Add a term in one of the termbase's languages.  This link is only active if there are no entries for one or more of the termbases's languages.

Remove Language

Remove terms in one of the termbase's languages.  This link is only active if the language is selected.  Click the language name to select the language.

Add Term

Add a term in one of the termbase's languages.

Remove Term

Removes the selected term.  Select the term by clicking on the term.

Add Field

 

to current field

Add a description of the current term, e.g. Term Type: Acronym.

after current field

Adds another description after the current field.

Edit Field

Edit the selected field.  Select a field by clicking on the field.  Alternately, double click a field to edit it.

Remove Field

Removes the selected field.
