Manage Input Models

Summary

Manage input models for a termbase.

Table

Displays all Input Models that have been added for this termbase.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select an input model by clicking the radio button next to the input model.

Buttons

 

Add

Add a new input model (see Create/Edit Input Model).

Edit

Edit the selected input model (see Create/Edit Input Model).

Remove

Remove the selected termbase.

Make Default

Make the selected input model the default for new termbase entries.

Unset Default

The default input model is no longer marked as default.

Done

Returns to the previous screen.
