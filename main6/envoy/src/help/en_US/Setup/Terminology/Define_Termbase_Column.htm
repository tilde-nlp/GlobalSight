Define Termbase Column

Summary:

Define the characteristics for a column in a delimited text file.

Column Name:

Identifies the column for display in previous screen - populated with column header.

Map Column to Type:

Identifies the type of information contained in this column.

 

Concept Status, Concept Domain, Concept Definition, Concept Source, all apply to the entire termbase entry.

 

Term, Usage, Term Type, Part of Speech, Gender, Number, Term Definition, Term Example, Source, apply to one of the columns (terms) in an entry.

Term Language:

Used for Terms only: specifies the language the term is in.

Associate with Column:

Specifies the column to which the information is related (usually a term).

Buttons

 

Cancel

Cancels the column mapping.

OK

Saves the column mapping settings.
