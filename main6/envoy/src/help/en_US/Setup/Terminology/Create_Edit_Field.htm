Create/Edit Field

Summary:

Define a field for a termbase.

Field Name:

For display in GlobalSight. This value cannot be edited after the object has been saved.  The name may only contain letters, numbers, the underscore "_" or the hyphen "-".  

Field Type:

Select a field type for this field.  Text fields contain simple text, attribute fields can be limited to certain values.  Other field types are described in more detail in the Explanation of Field Type: box.

Field Format::

Non-Editable: Describes the Field Format as text or attribute.

System Field:

Non-Editable: Indicates whether the field is a system field (checked) or not (unchecked).

Explanation of Field Type:

Non-Editable: Describes the field type.

Allowed Values (for attribute fields):

A comma separated list of values for an attribute.  Termbase entries which use this field must use one of the values in this list.  Does not apply to text fields.

Suggested Attribute Values

Suggested values are provided for some field types.  Copy these values into the Allowed Values box to use them for this field.

Buttons

 

(click to copy):

Copies the values from Suggested Attribute Values into Allowed Values (for attribute fields):.

Ok

Saves the field and closes the window.

Cancel

Closes the window discarding any changes.
