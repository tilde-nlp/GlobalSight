Create/Edit Input Model

Summary

Define basic structures for termbase entries.

Name:

Enter a name for this input model.

Field

Lists languages and all associated fields and properties.  Select any item by clicking on it.

Buttons

 

Save and Close

Saves the input model and closes the window.

Close

Closes the window discarding any changes.

Add Language...

Adds a language to the input model (see Add Language Constraint).

Remove Language

Removes the selected language.

Add Synonym

Adds a synonym to the input model (see Add Synonym Constraint).

Remove Synonym

Removes the selected synonym.

Add Field: to current field...

Adds a field as a child (sub-field) of the currently selected field (see Add Field Constraint).

Add Field: after current field...

Adds a field as a sibling (same level) of the currently selected field (see Add Field Constraint).

 

Edit Field

Edits the selected field.

Remove Field

Removes the selected field.
