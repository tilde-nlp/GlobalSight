Create/Edit Translation Memory

Summary:

Create or modify a TM in which to store translations.

Name*:

For display in GlobalSight. This value cannot be edited after the object has been saved.  The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Domain:

Optional description: the area of use for terms in this TM.

Examples: IT, Medical, Educational.

Organization:

Optional description: the organization for which terms in this TM are created.

Examples: Marketing, Sales, Engineering.

Description:

Optional description for this TM.

Index Target:

If checked, when import TM or populate TM, system will create lucene index file on hard disk for both source and target TM data. Default unchecked, source only for performance purpose.

TU Attributes:

Add/edit TU Attributes for TM, add permission "Setup -> Translation Memory -> Enable TM Attributes" to view it.

Buttons

 

Save

Saves the new/changed TM.

Cancel

Cancel TM creation/edit.
