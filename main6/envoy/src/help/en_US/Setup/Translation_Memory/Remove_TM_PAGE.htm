Removing Translation Memory

Summary:

Remove entire TM(s) or TU(s) and TUV(s) from TM(s).

Select entries to remove

Entire TM: Remove the selected TM(s) from GlobalSight.

By language: Remove all TUV entries of the specified language in the selected TM(s).

By TU Listing (TMX): Remove all TU entries from the TU listing file (TMX) in the selected TM(s).

Buttons

 

Cancel

Cancels the TM removing.

OK

Start removing action and move to the next screen of the TM Remove ( Remove TM Status).
