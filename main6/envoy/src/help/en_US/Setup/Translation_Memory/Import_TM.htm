Import Translation Memory

Summary:

Import entries from an exported translation memory in the form of an XML file.

Select a file to import:

Use the Browse... button to select an XML translation memory from your local machine or shared drive.

Import file format:

G-TMX: Imports entries from a GlobalSight native format XML file.

TMX (level 1): Imports entries from a TMX level 1 compatible XML file.

TMX (level 2): Imports entries as a TMX level 2 compatible XML file.

Trados© TMX (RTF):  Imports entries from a Trados© compatible XML file containing TM data for RTF files.

Trados© TMX (HTML):  Imports entries from a Trados© compatible XML file containing TM data for HTML files.

Trados© TMX (FrameMaker):  Imports entries from a Trados© compatible XML file containing TM data for FrameMaker files.

Trados© TMX (QuarkXPress):  Imports entries from a Trados© compatible XML file containing TM data for QuarkXPress files.

Buttons

 

Cancel

Cancels the TM import.

Next

Moves to the next screen of the TM import (Set Import Options).
