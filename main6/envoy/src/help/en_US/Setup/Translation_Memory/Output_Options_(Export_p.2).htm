Output Options (Export p.2)

Summary:

Select whether System Fields should be output with the TM export.

Output Options

Displays the number of entries that will be exported.

Output System Fields

Checked: System fields are written to the exported TM.
Un-checked: System fields are not written to the exported TM.

Buttons

 

Cancel

Cancels the TM export.

Previous

Returns to the previous screen.

Export...

Proceeds with TM export using the selected options (see Exporting...).
