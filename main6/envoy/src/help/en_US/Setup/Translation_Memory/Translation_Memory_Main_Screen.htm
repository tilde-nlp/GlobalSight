Translation Memory Main Screen

Summary:

Create a translation memory to store translations

Table

Displays all translation memories that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a TM by clicking the radio button next to the TM.

Buttons

 

Corpus Browser

Opens the corpus browser for the selected TM.

Statistics

Displays basic information about the selected translation memory (TM).  A TU (translation unit) is a segment in the TM, and a TUV (translation unit variant) is a locale-specific entry (translation).  Totals are given for the entire TM, as well as each locale for which there are entries in the TM.

Maintenance...

Search and replace entries in the selected TM.  Maintenance can also be used to perform a concordance search within the TM.

Import...

Imports translation memory data into the selected TM (see Import TM).

 

Export...

Exports the selected translation memory (see Export TM).

Reindex...

Reindex selected TM or All TMs (see Reindex TM)

Duplicate...

Creates a copy of the selected translation memory (opens the Create/Edit Translation Memory screen for the new translation memory).

Edit...

Edit the descriptions for the selected translation memory (see Create/Edit Translation Memory).

Remove

Removes the selected TM from the system. (see Removing Translation Memory)

New...

Add a new translation memory to the system (see Create/Edit Translation Memory)..

Users

Modify the users who can access this TM, only Administrator user can see this button.
