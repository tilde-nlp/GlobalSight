Results

Summary:

Search the target locale entries for the text in the Replace box and replace is with the text in the With box.

Segments Found

Displays the number of entries found whose text matches the search text.

Replace:

This text is searched for in the target locale entries.

Button
Highlight

 

Highlights the text in the Replace box within the entries under Target locale.

In Target Locale:

Displays the target locale in which the replacements will be made.

With

Enter the text to replace the word(s) in the replace box.

Match case:

Checked: Only entries whose case matches the replace text are replaced.

Unchecked: All entries whose text matches the replace text are replaced.

Check All | Clear All

Checks or clears all of the entries on this page of search results.

Check All Pages | Clear All Pages

Checks or clears all of the entries on all pages of search results.

 

Use the checkboxes to select the entries that should be searched for text in the Replace box and replaced with text in the With box.

1-10 of X

Use these buttons to navigate through additional pages of results.

Buttons

 

Cancel

Cancels the search and returns to the TM Management screen.

Previous

Returns to the previous screen.

Replace

Performs the replace operation as specified above.  All checked entries are searched for the term in the Replace box and that text is replaced with the text in the With box.

Delete Target

Remove the target from the TM

Delete Whole Segment

Remove the entire segment from the TM.
