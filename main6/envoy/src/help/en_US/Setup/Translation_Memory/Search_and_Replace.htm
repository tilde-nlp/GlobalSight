Search and Replace

Summary:

Search for entries in the TM in order to make changes

Find:

Enter the search term.

In Source Locale:

Select the locale in which to search.  This does not have to be the same locale in which you will ultimately replace entries.

Match case

Checked: Only entries whose case matches the search text are displayed.

Unchecked: All entries whose text matches the search text are displayed.

Show target locale:

Select the local in which you plan to make changes.

Buttons

 

Cancel

Cancels the TM maintenance.

Next

Moves to the next screen of the TM maintenance (Results).
