Reindex TM

Summary:

Reindex the selected TM(s).

Select TM to reindex:

Use radio button to select all TMs or selected TM.

Index Target:

If checked, system will create lucene index file on hard disk for both source and target TM data. If unchecked, only for source.

 

Buttons

 

Cancel

Cancels the reindexing.

Next

Reindex the TM (see Reindexing TM).
