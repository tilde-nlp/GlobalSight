Export TM

Summary:

Export all entries from a translation memory.  The exported translation memory can be later imported by GlobalSight or by another program that utilizes translation memory.

Select entries to export:

Entire TM: Exports all TUVs from the TM.
By language: Exports only TUs containing TUVs from the selected locale from there TM.

TM attribute:

Select the attribute name and value from the list.

Add : Add the attribute you selected.

Remove : Remove the attributes you selected from added area.

By language:

According language filter query.

Projects:

According project filter query.

TU ID:

According tu id filter query.Format example:201,202,205-207,210-220.

Job ID:

According job id filter query.Format example:201,202.

String ID:

According sid filter query.

Is Regex : SID whether regular expressions.

Creation User:

According create User filter query.

Modification User:

According modify User filter query.

Creation Date:

According creation date filter query.

start : Greater than equal to the start time.

end : Less than equal to the end time.

Modification Date:

According modify date filter query.

start : Greater than equal to the start time.

end : Less than equal to the end time.

Last Usage Date:

According last usage date filter query.

start : Greater than equal to the start time.

end : Less than equal to the end time.

Export format:

G-TMX: This is the GlobalSight native format and is the best choice if the TM is to be be re-imported into another GlobalSight installation.  The export is an XML file.

TMX (level 1): Exports entries as a TMX level 1 (1.4b) compatible XML file.

TMX (level 2): Exports entries as a TMX level 2 (1.4b) compatible XML file.

Trados© TMX: Exports entries as a Trados© compatible XML file.  You must choose this option if you wish to use the TM with Trados.

File Encoding:

The default value of UTF-8 is correct.

Buttons

 

Cancel

Cancels the TM export.

Next

Moves to the next screen of the TM export (Output Options).
