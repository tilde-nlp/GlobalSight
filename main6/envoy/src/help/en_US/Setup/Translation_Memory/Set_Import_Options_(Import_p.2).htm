Set Import Options (Import p.2)

Summary:

Select options for existing TUs during the TM import.

Set Import Options

TUs are compared based on their source TUVs. Select one of the following options:

Merge TUs with existing TUs: Matching New TUVs are merged (added) to existing TUVs.

Overwrite existing TUs: Matching New TUVs overwrite existing TUVs.

Discard TUs: Matching New TUVs are discarded (not added) to existing TUVs.

 

Definitions:

New TUV: a TUV in the file you are importing

Existing TUV: a TUV in the TM already in the system

Matching TUV: New and Existing TUVs are compared against one another based on the source TUV for each.  They are considered a match if their source TUVs are the same.  The options above only describe what happens to matching TUVs.  New TUVs that are different from all existing TUVs are always added to the TM.

Buttons

 

Previous

Returns to the previous screen.

Cancel

Cancels the TM import.

Import

Proceeds with TM import using the selected options.
