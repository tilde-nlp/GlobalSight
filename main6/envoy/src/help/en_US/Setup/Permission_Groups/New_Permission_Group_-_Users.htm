New Permission Group - Users

Summary

Assign users to permission group.

Left Column (Available)

Lists all available users.  Select a user by clicking on one, shift and ctrl can be used in combination with a click to select multiple users.

Right Column

(Added)

Lists all users added to the permission group.  Select a user by clicking on one, shift and ctrl can be used in combination with a click to select multiple users.

Buttons

 

>>

Add selected user to the permission group.

<<

Remove selected user from the permission group.

Cancel

Cancel the creation of the permission group.

Previous

Returns to New Permission Group - Permissions.

Save

Saves new permission group.
