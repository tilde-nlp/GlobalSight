New Permission Group - Permissions

Summary:

This is where permissions can be added/removed from specific permission group.

Permission Tree:

Lists all permissions that can be added/removed from permission groups.  Use the or to expand/collapse permissions.  Use the checkbox, to the right of the permission, to add/remove the permission.  

Note: Selecting a permission will select all sub-permissions.  Removing a permission will remove all sub-permissions.

Buttons

 

 

Expand All

Expand all permissions

Collapse All
