Export Error

Summary:

View export errors and attempt to export again.

Job:

This is the name given to the job by the initiator or automatically generated by the system.

Source Locale:

The source locale of the content in this job.

Target Locale:

The target locale of the content in this job.

Project:

Name of the project.

Example: World Services R&D.

Localization Profile:

The localization profile used to create this job.

This determines the workflows and their activities for this job.

Date Created:

Date and time that the job was created.

Data Source:

The file profile used to import the content in these pages.

Page Table

Lists all pages (files) in the job.  Select a page by clicking the checkbox next to the page name.

Message

The message may contain useful information about why the error occurred.

Buttons

 

Previous

Return to the previous screen.

Re-Export

Export the selected page.
