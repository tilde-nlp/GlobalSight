Workflow Details

Summary:

Allows the user to view a workflow, enter a planned completion date, and view secondary target files.

Graphical Workflow

Shows the sequence of activities for the workflow.  Completed activities are colored dark blue, the current activity is colored green, and future activities are colored light blue.

View Estimated Completion Date

Select an activity to display all qualified users for that activity in the locale pair of this workflow and the estimated time for each user to complete the activity.  If the selected activity is not in the default path, the a message informs the user that the activity is not in the default path.

Workflow History

Displays details of completed and current activities for the workflow.

Buttons

 

Save

Saves the planned completion date.

Close

Closes the workflow details screen and returns to job details.
