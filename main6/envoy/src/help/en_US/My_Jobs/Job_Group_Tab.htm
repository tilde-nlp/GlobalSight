Job Groups

Summary:

This screen lists all job groups.

Table

Displays all job groups.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a job group by clicking the checkbox next to the job group.  View the job list by clicking on the job group name.

Inputs

 

Group Id:

Job group whose job group id equal with the text in the blank text field.

Group Name:

Search job groups whose group name contians the text.

Project

Search job groups whose project name equal with the selected.

Source Locale

Search job groups whose source locale name equal with the selected.

Buttons

 

New...

Creating a new job group.

Remove...

Delete the job group.
