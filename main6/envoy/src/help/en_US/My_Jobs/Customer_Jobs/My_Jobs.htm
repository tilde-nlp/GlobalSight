My Jobs

Summary:

Displays jobs currently in the process of localization.

Job Name:

Search for jobs whose names contain the text entered in the blank text field.

Table

Displays all in jobs, or those that matched the search criteria.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a job by clicking the radio button next to the job.

Buttons

 

Download...

Download any finished files (see Download Files)

View File List...

View the status of all files in a job (see View File List).

Comments...

View comments for this job (see Segment Comments).

Assign...

Assign a review user for this job (see Assign User).
