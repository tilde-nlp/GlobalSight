Change Workflow Managers

Summary:

Change the Workflow Managers assigned to each workflow.

Workflows:

Displays all workflows for this job, listed by target locale.  Select a workflow by clicking on it.

Workflow Managers:

Displays all workflow managers in GlobalSight.  Select a workflow manager by clicking on the user's name.  Select multiple workflow managers by holding down the Ctrl key and clicking each user's name.

Buttons

 

Revert

Reverts to the last saved workflow managers for this workflow.

Revert All

Reverts all workflows to their last saved workflow managers for this job.

Cancel

Cancels any unsaved changes and returns to the previous screen.

Save All

Saves changes made to workflow managers for all workflows

Done

Returns My Jobs. If there are unsaved changes, you are prompted to save them.
