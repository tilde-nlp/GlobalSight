Detailed Word Count

Summary:

View detailed word counts to see how activity was leveraged.

Tables

Displays word counts for all pages in the activity.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.

Leverage Match Threshold (LMT)

The LMT value set in the TM profile.  Segments below the LMT are considered No Match.  Segment between 99% and the LMT are considered Fuzzy matches (100 > x >= LMT).

Repetitions

A Repetition is a segment for which match percentage is not 100%, and repeats within a file or job.  Each Repetition, regardless of the number of times it is repeated, is added once to the repetition total.
