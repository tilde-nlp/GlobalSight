PDFs Tab

Summary:

View PDF information about this job.

Target Locale:

The target locale of workflow.

Word Count:

The total word count of workflow.

PDF File Number:

Display the Existed PDF File Number and Total PDF File Number.
The format is {The Existed PDF File Number}/{The Total PDF File Number}.

Process Status:

The Status for the process of creating PDF, which has 3 value:
"None": Default value.
"In Progress": The process of creating PDF files is started.
"Done": The process of creating PDF files is done.

Buttons:

Create PDF

Start the process of creating PDF files.

Download

Download the existed PDF files.

Reset

Stop the process of creating PDF files.
