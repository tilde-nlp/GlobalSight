Jobs - Search

Summary:

Filter jobs displayed on the Jobs Screens.

Job Name:

begins with: Displays jobs with names beginning with the text in the adjacent blank text field.

ends with: Displays jobs with names ending with the text in the adjacent blank text field.

contains: Displays jobs with names Job name containing the text in the adjacent blank text field.

 

blank text field: Search matches the text in this box to job names and filters the jobs displayed in the search results.

Job ID:

equals: Displays jobs with an ID equal to the value in the adjacent blank text field.

less than: Displays jobs with an ID less than the value in the adjacent blank text field.

greater than: Displays jobs with an ID greater than the value in the adjacent blank text field.

 

blank text field: Search matches the value in this box to job ids and filters the jobs displayed in the search results.

Status:

Displays jobs with the selected status in the drop-down.

Project:

Displays jobs belonging to the selected project in the drop-down.  If no project is selected, project is not used as a filter criteria.

Source Locale:

Displays jobs with the selected source locale in the drop-down.  If no source locale is selected, source locale is not used as a filter criteria.

Target Locale:

Displays jobs containing workflows with the selected target locale in the drop-down.  If no target locale is selected, target locale is not used as a filter criteria.

Priority

Displays jobs with the selected priority in the drop-down.  If no priority is selected, priority is not used as a filter criteria.

Creation Date Range

Displays jobs that were created during the time range specified.  Time ranges are relative to the current time, e.g. from 5 hours ago (start) until 1 hour ago (end).

 

Starts:

blank text field: Enter a numeric value.

dropdown: Select a time increment for the numeric value.

 

Ends:

blank text field: Enter a numeric value.

dropdown: Select a time increment for the numeric value.  Selecting "now" ignores the value in the preceding blank text field.

Estimated Completion Date Range

Displays jobs that have an estimated completion date that falls during the time range specified.  Time ranges are relative to the current time, e.g. from 5 hours ago (start) until 1 day from now (end).

 

Starts:

blank text field: Enter a numeric value.

dropdown: Select a time increment for the numeric value.

 

Ends:

blank text field: Enter a numeric value.

dropdown: Select a time increment for the numeric value.  Selecting "now" ignores the value in the preceding blank text field.

Buttons

 

Search

Displays jobs matching the selected criteria.

Clear

Removes all entered and selected search criteria from the screen.
