Localized Jobs

Summary:

Displays jobs with completed workflows (all activities finished).

Search

 

Status:

Select a job status (Pending, Ready, In Progress, Localized, Exported, Archived,All Status) to filter the jobs displayed in the search results.

Advanced.../Simple...

Click on Advanced.../Simple... to change search mode.
In Advanced search mode, you can search jobs by Creation Date and/or Estimated Completion Date. Select Exported from Status list, then you can search job by Exported Date .

Table

Displays all localized jobs that match the search criteria.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a job by clicking the checkbox next to the job.  View the job details by clicking on the job name.

Inputs

 

Group Id:

If the job in the group, and will display the group's id .

Job Id:

=: Job whose job id equal enter a value or between enter interval value, for example 1,3,5,6-9

>: Job whose job id greater than the text in the blank text field.

<: Job whose job id less than the text in the blank text field.

(Blank Text Field)

Search matches the text in this box to job id and filters the jobs displayed in the search results.

Job Name

Search jobs whose job name contians the text.

Project

Search jobs whose project name equal with the selected.

Source Locale

Search jobs whose source locale name equal with the selected.

Buttons

 

Search

Searches based on the criteria entered.

Export...

Initiates the export process from GlobalSight. This will move the selected job to the Exported status.

Export Source...

Exports the source file in this job. The source files will contain reference markers for all added and deleted content.  The job will be canceled.

Discard

Permanently deletes the job from the System. All translations done up to this point are lost.

Download...

Select Files to Download

Export & Download...

Exports and Download the files in this job.
