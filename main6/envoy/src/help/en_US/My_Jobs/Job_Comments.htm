Comments

Summary:

View all comments associated with this job.

Job Comments

View and add comments that will be visible to all participants in all workflows.

Buttons

 

New...

Launches the add comment window for Job Comments (see Add/Edit Comment).

Edit...

Opens the selected Job Comment for editing.

Activity Comments

Shows all comments that have been added through individual activities in target locale workflows.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Use the Show: drop-down to filter the comments by target locale.

Segment Comments

Shows all comments that have been added to individual segments.  Click Target Locale to sort the table based on target locale; click a second time to reverse the sort order.  Use the Show: drop-down to filter the comments by target locale.
