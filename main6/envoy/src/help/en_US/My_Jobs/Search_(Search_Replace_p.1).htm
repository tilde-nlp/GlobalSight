Search (Search/Replace p.1)

Summary:

Search for entries in the Job in order to make changes

Target Locales:

Select the target locales in which to search.  Select a locale by clicking on the name of the locale.

Search:

Enter the string to search for.

Match case

Checked: Only entries whose case matches the search text are displayed.

Unchecked: All entries whose text matches the search text are displayed.

Buttons

 

Cancel

Cancels the Search/Replace.

Search

Searches for the specified string in the selected target locales (see Moves to the next screen of the TM maintenance (Search Results/Replace).
