Pending Jobs

Summary:

Displays jobs not ready for dispatch.

  • Make Ready moves a pending job to the Ready.

  • Discard removes a job from GlobalSight entirely.

  • If an error occurs during import, Examine Error shows any errors (See Import Error).

Search

 

Status:

Select a job status (Pending, Ready, In Progress, Localized, Exported, Archived,All Status) to filter the jobs displayed in the search results.

Advanced.../Simple...

Click on Advanced.../Simple... to change search mode.
In Advanced search mode, you can search jobs by Creation Date and/or Estimated Completion Date. Select Exported from Status list, then you can search job by Exported Date .

Table

Displays all pending jobs that match the search criteria.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a job by clicking the checkbox next to the job.
